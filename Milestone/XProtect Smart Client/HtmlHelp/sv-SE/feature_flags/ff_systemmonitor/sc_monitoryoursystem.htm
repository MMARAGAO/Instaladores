Övervaka ditt system

Fliken Systemövervakaren ger dig den befintliga statusen för dina servrar, anslutna enheter och datorn som kör XProtect Smart Client.

För mer information, se Fliken Systemövervakaren (förklaring).

Övervaka klientresurser

Antalet kameror i vyn tillsammans med upplösning, bildfrekvensEtt mått som påvisar mängden information i en rörlig videoupptagning. Anges i regel med FPS (Frames Per Second, bilder per sekund). och codecTeknik som används för att komprimera och dekomprimera ljud- och videodata, till exempel i en exporterad AVI-fil. resulterar i en belastning på datorn som kör XProtect Smart Client. För att se den befintliga belastningen på CPU, RAM-minne, och NVIDIA GPU-resurser:

  1. Klicka och dra fliken Systemövervakaren för att lossa den och placera den i ett separat fönster.
  2. Välj Denna dator.
  3. För att övervaka belastning i den aktuella vyn, välj fliken Live eller fliken Uppspelning.

Om din klientdator har extra NVIDIA-skärmadaptrar installerade kommer belastningen på dessa GPU:er även att visas.

Om belastningen är för hög kan du lägga till GPU-resurser till din dator genom att installera flera NVIDIA-skärmadaptrar. Milestone rekommenderar att du inte använder dig av konfigurationen Skalbart länkgränssnitt (Scalable Link Interface, SLI) i dina NVIDIA-displayadaptrar.

Fliken Systemövervakaren med Milestone Federated Architecture (förklaring)

Om du kör Milestone Federated Architecture™ är fliken Systemövervakaren uppdelad i två delar:

  • Ett fält visar en hierarkisk trädstruktur som representerar din distribuerade arkitektur
  • Det andra fältet är ett webbläsarbaserat område med relevanta systemdata för den valda servern.

Klicka på en server i platsfönstret för att se dess systemdata.

Om du flyttar ifrån fliken eller loggar ut från systemet och kommer tillbaka, minns fliken Systemövervakaren vilken server som är vald i din distribuerade arkitektur och fortsätter visa systemdata från denna server.

Du kan dra fliken Systemövervakaren till ett oberoende fönster för att övervaka flera servrar.
