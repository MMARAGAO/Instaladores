Bokmärken (konfiguration)

Denna funktionalitet är bara tillgänglig i vissa XProtect VMS-system. För information om funktionalitet i ditt system, se Skillnader mellan övervakningssystem.

Aktivera detaljerade bokmärken

För att kunna ge bokmärken ett namn och en beskrivning och ändra standardtidsintervall måste du aktivera information.

Gör så här:

  1. Öppna fönstret Inställningar.

  2. Välj fliken Funktioner.
  3. För att aktivera detaljerade bokmärken för direktsänd video, välj Lägg till bokmärkesinformation i listan Bokmärke för fliken Live.
  4. För att aktivera detaljerade bokmärken för inspelad video, välj Lägg till bokmärkesinformation i listan Bokmärke för fliken Uppspelning.
  5. Klicka på Stäng.
