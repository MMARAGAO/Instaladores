XProtect Access (konfiguration)

Lägg till inpasseringsövervakning till vyer

Du börjar med att definiera ett visningselement för passersystemet:

  1. På fliken Live (se Fliken Live (förklaring)), i inställningsläge, välj den vy du vill använda för att övervaka passersystemet.
  2. I rutan Systemöversikt, klicka på Inpasseringsövervakning och dra den till ett visningselement.
  3. Ange inställningarna i dialogrutan Inställningar för inpasseringsövervakning (se Inställningar för inpasseringsövervakning) som visas. När du har valt en dörr kan du behålla standardinställningarna eller ändra dem efter behov.
  4. Klicka på OK. Inpasseringsövervakning läggs till i vyn.

En händelse kan användas till exempel för att se till att en kamera spelar in med en viss bildfrekvens, för att aktivera utdata, för att skicka e-post eller en kombination av dessa. visas detta till höger i visningselementet.

Inställningar för inpasseringsövervakning

Ange följande inställningar för inpasseringsövervakning:

Namn

Beskrivning
Dörr

Ange den dörr du vill visa passersystemhändelser från. När du väljer en dörr visas återstående inställningar i dialogrutan med aktuella värden.
Källor

Välj vilken typ av passersystemkällor du vill ta emot händelser från. Listan kan t.ex. innehålla dörrar eller specifika inpasseringspunkter för en dörr. En inpasseringspunkt är en plats där man kan ta sig in, inklusive associerade fysiska enheter som t.ex. kortläsare, knappsatser, sensorer eller knappar. En dörr har normalt två inpasseringspunkter, som styr ingång respektive utgång genom dörren.

Listan över källor konfigureras av systemadministratören.

Välj den kamera från vilken du vill visa video som hör till dörren. Som standard listar systemet de kameror som systemadministratören har associerat med vald dörr, men du kan också välja någon annan kamera i systemet.

Ändra inställningar för inpasseringsövervakning

På fliken Live kan du ändra inställningar för inpasseringsövervakning:

  1. Klicka på Inställning och välj det visningselement du vill ändra.
  2. I rutan Egenskaper klicka på knappen Inställningar för inpasseringsövervakning.
  3. Ange inställningarna i dialogrutan Inställningar för inpasseringsövervakning (se Inställningar för inpasseringsövervakning) som visas.
  4. Klicka på OK för att stänga dialogrutan, och sedan på Inställning för att återgå till livevisning.

Anpassa din vy

Med overlay-knapparKnapp som visas som ett lager ovanpå videon när du flyttar muspekaren över enskilda kamerapositioner i vyer under fliken Live. Med overlay-knappar kan du aktivera högtalare, händelser, utgångar, flytta PTZ-kameror, starta inspelning, ta bort signaler från kameror osv. kan du anpassa ditt gränssnitt. Du kan lägga till overlay-knappar för passersystem till ett visningselement från en lista med kommandon som konfigurerats för dörrarna eller inpasseringspunkterna.

Exempel på användning:

  • Ha direkt åtkomst till kommandoknappar i andra visningselement än inpasseringsövervakning
  • Placera kommandoknapparna direkt vid en dörr i visningselementet
  • Lägg till andra kommandoknappar än de som angetts i Inställningar för inpasseringsövervakning

Gör så här:

  1. Klicka på Inställning på fliken Live och välj det visningselement du vill ändra.
  2. I rutan Overlay-knappar, klicka på Passersystem.
  3. Leta upp det kommando du vill lägga till och dra det till ett visningselement.
  4. Klicka på Inställning för att återgå till livevisning.

    Overlay-knappen visas när du flyttar muspekaren över visningselementet.

Hantera information om kortinnehavare

Om ditt passersystem har konfigurerats för det kan du gå direkt till en webbsida med en representation av kortinnehavarens post och t.ex. utföra användaradministration eller få mer information om kortinnehavaren.

Om plugin-programmet har stöd för djuplänkning finns följande förutsättningar för passersystemet:

  • Det måste innefatta en webbklient
  • Det måste ha stöd för djuplänkning.

Så här hanterar du information om kortinnehavare:

  1. På fliken Passersystem väljer du listan Kortinnehavare.
  2. Sök efter en kortinnehavare och välj personen i listan.
  3. Till höger, under informationen om kortinnehavaren, kan du klicka på en länk till t.ex. en webbsida. Beroende på plugin-programmet kan det finnas stöd för flera länkar, och du kan bli tillfrågad om ytterligare autentiseringsuppgifter.
  4. Du kan redigera flera funktioner, inklusive information om kortinnehavare och passagebehörigheter.
  5. I det här exemplet stänger du webbsidan och återgår till XProtect Smart Client.

Aktivera och inaktivera aviseringar om passagebegäran

Du kan inaktivera hantering av passagebegäran, t.ex. om endast en person ska hantera passagebegäran.

  1. Klicka på och sedan Inställningar för att öppna fönstret Inställningar.
  2. Välj Passersystem och inaktivera aviseringar om passagebegäran.

Du kan återaktivera aviseringar för passagebegäran om du behöver hantera passagebegäran igen. Du kan också ändra alternativen för passersystemet genom att klicka på ikonen Inställningar inuti en avisering om passagebegäran.

Om fältet Följ server är markerat så kontrollerar din systemadministratör inställningen Visa aviseringar om passagebegäran.
