XProtect LPR (konfiguration)

Lägg till nummerskyltskameror i vyer

  1. I Inställningsläget på fliken Live väljer du den flik där du vill lägga till en nummerskyltskamera.
  2. I rutan Systemöversikt klickar du på Nummerskyltsigenkänning (LPR) och drar den till relevant visningselement.
  3. I dialogrutan Välj nummerskyltskamera, expandera önskad server för att visa en lista med tillgängliga nummerskyltskameror på den servern.

Du kan ange hur du vill visa händelser från nummerskyltskameror på fliken Live i rutan Egenskaper (se Justera vyinställningar för nummerskyltsigenkänning (LPR)).

Justera vyinställningar för nummerskyltsigenkänning (LPR)

  1. Klicka på Inställning på fliken Live.
  2. I Egenskaper, bredvid Nummerskyltskamera, klicka på knappen Bläddra för att öppna dialogrutan Välj nummerskyltskamera och välj en annan nummerskyltskamera.
  3. Välj ordning för LPR-händelser i din lista till höger om förhandsvisningen:
    • Nyaste överst: Visa de senaste LPR-händelserna överst i listan
    • Nyaste nederst: Visa de senaste LPR-händelserna längst ned i listan
  4. Om du vill visa listan med nummerskyltar från en kamera men vill visa video från en annan väljer du en annan kamera i namnet Kameranamn.

Aktivera LPR-serverstatus på kartor

Det går att visualisera LPR-servrar på kartor och få deras aktuella status visad på kartorna. Aktivera LPR-serverstatus på kartor:

  1. Klicka på Inställning på fliken Live.
  2. Välj relevant karta i Vyer.
  3. Högerklicka på kartan och välj Verktygslåda.
  4. I verktygslådan, klicka på ikonen Lägg till plugin-element för att öppna fönstret Elementväljare.
  5. Välj relevant LPR-server och dra den till kartan.
  6. Högerklicka på LPR-serverikonen på kartan och välj Statusdetaljer, för att få livestatus för LPR-servern och de nummerskyltskameror som hör till servern.

Du kan associera den LPR-specifika kartan med din Larmlista genom att lägga till kartan på fliken Larmhanteraren.

Aktivera LPR-specifika element

För att kunna se all relevant information rörande nummerskyltsigenkänning (LPR) i din XProtect Smart Client måste du göra följande på fliken Larmhanteraren:

  1. Gå till fliken Larmhanteraren. I listan Larm högerklickar du på ikonen Bild bredvid kolumnen Snabbfilter. Välj följande på menyn: Objekt, Tagg och Typ.

  2. Nu visar Typ alla händelser som är relaterade till LPR, Tagg visar landskoder och Objekt registreringsnummer för de registrerade fordonen.
