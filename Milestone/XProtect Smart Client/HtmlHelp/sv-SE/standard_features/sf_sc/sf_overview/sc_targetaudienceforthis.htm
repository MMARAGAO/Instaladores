Om denna handbok

Denna användarhandbok är främst för XProtect Smart Client-operatörerEn professionell användare av ett XProtect-klientprogram., men även för systemadministratörer och integratörer som ansvarar för att konfigurera, underhålla och felsöka XProtect Smart Client. Huvuddelen av konfigurationen sker dock i XProtect Management Client. För mer information, se administratörshandboken för XProtect VMS.

När denna manual refererar till gränssnittselement, utgår den ifrån att du använder ett gränssnitt som är visuellt justerat vänster-till-höger. Se även Höger-till-vänsterspråk (förklaring).

Vad finns vid de olika avsnitten i handboken?

Översikt

Översiktsinformation om XProtect Smart Client och dess funktioner, inklusive nyheter i den aktuella utgåvan och tilläggsprodukter.

Licensiering

Allt du behöver veta om licenser i XProtect Smart Client.

Krav och överväganden

Saker du bör vara medveten om innan du installerar XProtect Smart Client, till exempel systemkrav.

Installation

Information om hur du installerar XProtect Smart Client.

Konfiguration

Det mesta av konfigurationen görs av din systemadministratör i XProtect Management Client, men det finns några element som du kan konfigurera själv i XProtect Smart Client, till exempel vyer och programinställningar.

Optimering

Saker du kan göra för att optimera användarupplevelsen, till exempel att aktivera hårdvaruacceleration.

Drift

Detta avsnitt handlar om hur funktionerna i XProtect Smart Client fungerar. Till exempel hur du spelar upp inspelad video, genomför avancerade sökningar samt exporterar videoinspelningar och stillbilder.

Felsökning

Ibland kan det uppstå problem, till exempel kan du få felmeddelanden. Prova avsnittet Felsökning om du inte kan få saker och ting att fungera, innan du kontaktar supporten.

Uppgradera

Information om när en uppgradering krävs.

Vanliga frågor

Har hittar du svaret på vissa av de vanligast ställda frågorna.

Om denna manual inte tillhandahåller den information du behöver, kontakta Milestone teknisk support.
