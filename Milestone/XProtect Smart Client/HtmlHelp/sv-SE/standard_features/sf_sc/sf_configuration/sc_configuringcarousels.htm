Karuseller (konfiguration)

Innan du kan använda karusellenEn särskild position för visning av videoupptagningar från flera kameror, en i taget, i en XProtect Smart Client-vy. måste du:

  1. Lägg till karusellen till en vy.
  2. Ange kamerorna du vill inkludera i karusellen.

Lägg till karuseller till vyer

KarusellerEn särskild position för visning av videoupptagningar från flera kameror, en i taget, i en XProtect Smart Client-vy. låter dig kontinuerligt visa alla kameror i den karusellhastighet som du anger.

  1. Klicka på Inställning för att aktivera inställningsläget.
  2. I rutan Systemöversikt, dra objektet Karusell till ett visningselement.
  3. I fönstret Karusellinställning:
    1. Gå till avsnittet Kameror.
    2. Hitta och dubbelklicka på varje kamera som du vill lägga till i karusellen.
  4. För att ange i vilken sekvens kamerorna visas i karusellen med listan Valda kameror bör du markera kamerorna och flytta dem uppåt eller nedåt.
  5. Ange hur många sekunder varje kamera ska visas i karusellen. Du kan ange ett värde för alla kameror eller individuellt för varje kamera.
  6. Klicka på OK för att stänga fönstret Karusellinställning.
  7. Klicka på Inställning igen så att inställningsläget avslutas. Dina ändringar har sparats.
  8. (valfritt) För att ändra inställningar för karusellen, i inställningsläget gå till rutan Egenskaper och klicka på Karusellinställning.

Redigera karusellinställningar

I karusellinställningarna kan du lägga till eller ta bort kameror från karusellen, ändra ordningen på kameror eller ändra tidsinställningar.

  1. Klicka på Inställning för att aktivera inställningsläget.
  2. Markera visningselementet med karusell.
  3. På vänster sida, scrolla ner till rutan Egenskaper.
  4. Klicka på Karusellinställning. Ett fönster visas.
  5. Gör de ändringar som krävs och klicka på OK.
  6. Klicka på Inställning igen så att inställningsläget avslutas. Dina ändringar har sparats.

Inställningarna Live dataström, Bildkvalitet, Bildfrekvens och Bevara bildformat i rutan Egenskaper tillämpas på alla kameror i karusellen.
