PTZ-förinställningar (konfiguration)

Beroende på ditt övervakningssystem (se Skillnader mellan övervakningssystem) kan du skapa, redigera och radera PTZ-förinställningar.

Lägga till PTZ-förinställningar

Du kan definiera ytterligare PTZ-förinställningarEn negativ eller farlig handling eller situation som skulle kunna leda till åverkan på, förlust av, eller störning av en organisations anställda, egendom, operationer, tjänster eller funktioner.:

  1. Välj den PTZ-kamera i vyn som du vill skapa en ny PTZ-förinställning för.
  2. I kameraverktygsfältet klickar du på PTZ-ikonen för att öppna PTZ-menyn.
  3. Klicka på Hantera PTZ-förinställningar för att öppna fönstret.

  4. Klicka på för att lägga till en ny förinställning.
  5. Välj PTZ-förinställningens post och ange ett nytt namn för PTZ-förinställningen.
  6. Använd PTZ-knapparna för att navigera till önskad position och klicka på Uppdatera förinställning för att spara.
  7. Använd pilarna för att flytta PTZ-förinställningen uppåt och nedåt i listan. Detta kan vara användbart om listan innehåller många förinställningar.

Redigera PTZ-förinställningar

Du kan göra ändringar till befintliga PTZ-förinställningarEn negativ eller farlig handling eller situation som skulle kunna leda till åverkan på, förlust av, eller störning av en organisations anställda, egendom, operationer, tjänster eller funktioner., såsom ändra namn eller ändra förinställd position:

  1. Välj den PTZ-kamera i vyn som du vill ändra PTZ-förinställning för.
  2. I kameraverktygsfältet klickar du på PTZ-ikonen för att öppna PTZ-menyn.
  3. Klicka på Hantera PTZ-förinställningar och välj PTZ-förinställningen i dialogrutan.
  4. För att redigera förinställningens namn, kontrollera att namnet för PTZ-förinställningen är markerat. Dubbelklicka på texten och skriv över det befintliga namnet.

  5. Om kameran inte är i korrekt position, använd PTZ-knapparna för att gå till önskad position och klicka sedan på Uppdatera förinställning för att spara.
  6. Använd upp- och nedpilarna för att flytta PTZ-förinställningarna i listan.
  7. Klicka på Stäng.

Ta bort PTZ-förinställningar

Om du vill radera en befintlig förinställning markerar du den och klickar på .
