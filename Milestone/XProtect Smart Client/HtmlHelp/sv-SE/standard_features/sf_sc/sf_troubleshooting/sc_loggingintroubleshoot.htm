Inloggning (felsökning)

Felmeddelanden och varningar

Din användarbehörighet tillåter inte att du loggar in nu. Användarrättigheter kan variera beroende på tid på dagen, veckodag, o.s.v.

Du har försökt att logga in under en tid när dina användarrättigheter inte var tillräckliga för att logga in. För att lösa detta problem:

Vänta tills du har tillåtelse att logga in. Kontakta din systemadministratör om du undrar över dina användarrättigheter.

Du har inte tillgång till någon del av programmet. Rådfråga systemadministratören.

Du har för närvarande inte rättigheter för åtkomst till någon del av XProtect Smart Client och därför kan du inte logga in. För att lösa detta problem:

Kontakta din systemadministratör som vid behov kan ändra dina behörigheter.

Programmet kan inte starta eftersom två (eller fler) kameror använder samma ID.

Detta felmeddelande visas bara i ett mycket ovanligt scenario där en säkerhetskopierad konfiguration från ett XProtect VMS-system av misstag används utan modifiering på ett annat XProtect VMS-system. Detta kan ha till följd att olika kameror har konflikt om en och samma identitet, och det kan i sin tur blockera din tillgång till XProtect VMS-systemet. Kontakta din systemadministratör om du ser ett sådant meddelande.

Auktoriseringen misslyckades: Du kan inte auktorisera dig själv.

Du har angett dina egna autentiseringsuppgifter i fältet Auktoriserad av. Du kan inte auktorisera dig själv. För att lösa detta problem:

Du måste kontakta den person som är behörig att auktorisera. Detta kan t.ex. vara din chef eller systemadministratören. Personen måste ange sina autentiseringsuppgifter för att auktorisera din inloggning.

Auktoriseringen misslyckades: Du har inte behörighet att auktorisera.

Du har försökt att auktorisera en användare utan att ha de rättigheter som krävs. För att lösa detta problem:

Be din systemadministratör att kontrollera att du har behörighet att auktorisera andra användare, eller be någon som har behörighet att auktorisera användaren.

Det gick inte att ansluta. Kontrollera serveradressen.

Det var inte möjligt att ansluta till XProtect VMS-server på den angivna serveradressen. För att lösa detta problem:

Bekräfta att du har skrivit in rätt serveradress. Prefixet http:// eller https:// och portnummer krävs som del av serveradressen (exempel: https://123.123.123.123:80, där :80 indikerar portnumret). Kontakta din systemadministratör om du är tveksam.

Det gick inte att ansluta. Kontrollera användarnamnet och lösenordet.

Det gick inte att logga in med angivet användarnamn och/eller lösenord. För att lösa detta problem:

Bekräfta att du har skrivit in användarnamnet korrekt och skriv sedan ditt lösenord igen för att försäkra dig om att det inte innehöll några fel. Användarnamn och lösenord är skiftlägeskänsliga. Det kan exempelvis vara skillnad på att skriva Amanda och amanda.

Det gick inte att ansluta. Maximalt antal klienter är redan anslutna.

Maximalt antal klienter som samtidigt kan ansluta till övervakningssystemets server har uppnåtts. För att lösa detta problem:

Vänta en stund innan du ansluter igen. Om tillgång till övervakningssystemet är akut, kontakta din systemadministratör, som har möjlighet att utöka antalet samtidigt anslutna klienter.

Anslutning via en gammal säkerhetsmodell. Du kan inte ansluta till webbsidan med den nyaste säkerhetsmodellen.

Händer om du försöker logga in på en server som inte har ett certifikat installerat. För att lösa problemet, kontakta din systemadministratör eller klicka på knappen Tillåt för att logga in med HTTP, ett nätverksprotokoll som inte använder ett certifikat.

Du har inte längre tillåtelse att göra detta

Detta inträffar om dina tidsberoende användarrättigheter inte längre tillåter dig att göra någonting som du tidigare har kunnat göra. Anledningen är att när du är ansluten till vissa typer av övervakningssystem, (se Skillnader mellan övervakningssystem), kan dina användarrättigheter variera beroende på tid eller veckodag, etc. Du kan därför mycket väl kunna utföra åtgärden igen senare.

På grund av övervakningssystemets inställningar kommer din XProtect Smart Client-session att avslutas inom [...]

Visas om din pågående session i XProtect Smart Client håller på att avslutas. När du är ansluten till vissa typer av övervakningssystem (se Skillnader mellan övervakningssystem), kan dina rättigheter att använda XProtect Smart Client bero på veckodag, tid, etc.

När detta sker visas ett meddelande med ett antal minuter eller sekunder innan sessionen avslutas, exakt tid definieras på övervakningssystemets server.

Ingen användaraktivitet har upptäckts nyligen. Din XProtect Smart Client-session kommer att avslutas inom [...]

Visas om du inte har använt XProtect Smart Client på ett tag (exakt tid anges på övervakningssystemets server). Din XProtect Smart Client-session avslutas av säkerhetsskäl.

Om detta sker visas meddelandet ett antal minuter eller sekunder innan sessionen avslutas. Exakt tid definieras på övervakningssystemets server.
