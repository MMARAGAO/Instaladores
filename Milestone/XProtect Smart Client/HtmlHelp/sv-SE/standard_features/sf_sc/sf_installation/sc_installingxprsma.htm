Installera XProtect Smart Client

Du måste installera XProtect Smart Client på din dator innan du kan använda det. Du laddar ned XProtect Smart Client från övervakningssystemets server och installerar det på din dator.

Milestone rekommenderar att du alltid använder den senaste versionen av XProtect Smart Client. Då vet du att du har tillgång till nya funktioner i ditt XProtect-övervakningssystem.

  1. Öppna din webbläsare och anslut till konfigurationsservern med URL- eller IP-adressen till servern.
  2. För en av följande:
    •   Lokal server (http://localhost/installation)
    •   IP-adress för fjärrservern (http://[IP_address]/installation)
  3. På sidan Välkommen klickar du på Språk och väljer det språk du vill använda. XProtect Smart Client Installationsguide startas.
  4. Följ installationsanvisningarna i guiden. Guiden föreslår en installationssökväg. Vanligen kan du använda den föreslagna installationssökvägen. Om du tidigare har använt tilläggsprodukter, kan det dock hända att sökvägen är inaktuell.
