Instalace aplikace XProtect Smart Client

Pokud chcete aplikaci XProtect Smart Client používat, musíte ji nejdříve nainstalovat ve svém počítači. Stáhněte si aplikaci XProtect Smart Client ze serveru zabezpečovacího systému a nainstalujte ji do svého počítače.

Milestone obecně doporučuje vždy používat nejnovější verzi aplikace XProtect Smart Client, abyste měli přístup ke všem novým vlastnostem a funkcím, které váš zabezpečovací systém XProtect nabízí.

  1. Otevřete prohlížeč a připojte se k serveru pro správu pomocí adresy URL nebo IP tohoto serveru.
  2. Zadejte jeden z následujících údajů:
    •   Místní server (http://localhost/installation)
    •   IP adresa vzdáleného serveru (http://[IP_address]/installation)
  3. Na stránce s uvítáním klikněte na položku Jazyk a vyberte požadovaný jazyk. Spustí se průvodce XProtect Smart Client nastavením.
  4. V průvodci postupujte v souladu s pokyny instalace. Průvodce navrhne cestu pro instalaci. Obvykle můžete použít navrhovanou cestu pro instalaci. Pokud jste však dříve použili nadstavbové produkty, tato cesta již nemusí být platná.
