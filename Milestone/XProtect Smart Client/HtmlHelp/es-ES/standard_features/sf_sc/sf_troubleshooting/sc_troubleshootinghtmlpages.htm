Páginas web (solución de problemas)

Estoy recibiendo un error de secuencia de comandos al añadir una página web a una vista

La página web utiliza secuencias de comando que no son compatibles con el navegador utilizado para procesar la página web. Puede resolver el problema de cambiar el Modo de visualización en las propiedades de la página web.

Estoy recibiendo un error de secuencia de comandos al cargar una vista que contiene una página web

La página web utiliza secuencias de comando que no son compatibles con el navegador utilizado para procesar la página web. Puede resolver el problema de cambiar el Modo de visualización en las propiedades de la página web.

He usado secuencias de comandos para añadir botones de navegación o imágenes en las que se puede hacer clic a mi página HTML, pero la página HTML no funciona como se pretendía. Tenga en cuenta lo siguiente:
  • ¿Ha establecido Modo de visualización a Compatibilidad? Solo el modo Compatibilidad soporta secuencias de comandos.
  • ¿Ha utilizado la sintaxis correcta en su código HTML?
  • ¿Está habilitada la activación de comandos HTML en XProtect Management Client o en el archivo Client.exe.config?
  • ¿El público destinatario tiene los derechos de usuario para acceder a ciertas cámaras, vistas, características o pestañas en XProtect Smart Client?
