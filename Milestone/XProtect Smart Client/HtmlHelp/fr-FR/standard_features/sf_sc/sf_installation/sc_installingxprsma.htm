Installer le XProtect Smart Client

Vous devez installer XProtect Smart Client sur votre ordinateur avant de pouvoir l'utiliser. Téléchargez XProtect Smart Client à partir du serveur du système de surveillance et installez-le sur votre ordinateur.

Milestone recommande de toujours utiliser la dernière version du XProtect Smart Client pour pouvoir accéder à toutes les nouvelles fonctionnalités et fonctions possibles fournies dans votre système de surveillance XProtect.

  1. Ouvrez votre navigateur et connectez-vous au serveur de gestion en utilisant l'URL ou l'adresse IP du serveur.
  2. Entrez l’un des éléments suivants :
    •   Serveur local (http://localhost/installation)
    •   L'adresse IP du serveur distant (http://[IP_address]/installation)
  3. Cliquez sur Langue dans la Page d'accueil et sélectionnez la langue souhaitée. L'assistant de configuration XProtect Smart Client démarre.
  4. Suivez les instructions d'installation de l'assistant. L'assistant propose un chemin d'installation. Normalement, il suffit de poursuivre et d'utiliser le chemin d'installation proposé. Toutefois, si vous avez déjà utilisé des produits complémentaires, ce chemin pourrait ne plus être valide.
