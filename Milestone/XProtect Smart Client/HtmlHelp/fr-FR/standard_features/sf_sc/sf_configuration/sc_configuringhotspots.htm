Hot spots (configuration)

Avant d'utiliser le hot spotEmplacement spécifique permettant de visionner des images de caméra agrandies et/ou de grande qualité dans les vues XProtect Smart Client., vous devez :

  1. Ajouter un hot spot à une vue. Voir Ajouter des hot spots à des vues.
  2. Spécifiez les paramètres du hot spot. Voir Paramètres du hot spot.

Ajouter des hot spots à des vues

Si votre vue contient un hot spot et que vous cliquez sur une caméra, le flux vidéo de la caméra s'affiche avec une résolution élevée dans l'élément de vue du hot spotEmplacement spécifique permettant de visionner des images de caméra agrandies et/ou de grande qualité dans les vues XProtect Smart Client..

Étapes :

  1. Cliquez sur Configuration pour entrer en mode configuration.
  2. Dans le panneau Vue d'ensemble du système, glissez-déposez l'élément Hot spot dans la position de vue requise. La position affiche une icône de hot spot : .
  3. Cliquez sur Configuration à nouveau pour quitter le mode configuration.
  4. (facultatif) Pour régler les propriétés du hot spot, rendez-vous sur le panneau Propriétés dans le mode de configuration.

Vous pouvez, afin d'économiser de la bande passante, indiquer une image de faible qualité pour les autres positions de votre vue et une qualité élevée pour le hot spot.

Paramètres du hot spot

Dans le panneau Propriétés (voir Paramètre des caméras), vous pouvez spécifier les paramètres pour le hot spotEmplacement spécifique permettant de visionner des images de caméra agrandies et/ou de grande qualité dans les vues XProtect Smart Client.. Les paramètres Flux en direct, Qualité d'image, Débit d'image et Conserver le rapport hauteur/largeur de l'image s'appliquent à l'ensemble des caméras dans le hot spot.

Pour faire apparaître les propriétés, vous devez sélectionner l'élément de vue, puis cliquer sur Configurer.
