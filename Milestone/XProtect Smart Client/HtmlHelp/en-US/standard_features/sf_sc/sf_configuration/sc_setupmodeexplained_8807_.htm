Setup mode (overview)

In setup mode, you can create views for your devices and other types of content, you can add overlay buttons, and set the properties for the cameras and other types of devices.

Number Description

1

When you enter setup mode, parts of the user interface are highlighted.

2

Create views and groups for your views. See also Create view groups or Create views.

3

Add cameras and other types of devices and content to your views. See also Add cameras and other elements to views.

4

Add overlay buttons to cameras to trigger auxiliary commands. See also Overlay buttons (explained).

5

Set the camera properties. See also Camera settings.
