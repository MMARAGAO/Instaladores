Check the status of your server connection

In the Status window, you can check the status of your server connection, for example, to see if you are using an older security model (HTTP) or the newest security model (HTTPS). If multiple sites are connected through Milestone Federated Architecture, you can also check the connected sites.

Connection statuses

  • HTTPS

  • HTTP

  • Not connected

Steps: 

  1. In the application toolbar, click the Status button. A window appears.

  2. On the Login information tab, check the status of your connection.

See also Status window (explained).
