Remove sequences from an incident project

If you have added irrelevant sequences to an incident project, you can remove them again.

  1. Select the Incidents tab and double-click the incident project that you want to update.

    2. If it is difficult to find a specific incident project, you can use the filter options. See Filter for incident projects.

  2. On the Sequence list tab, select the sequence you want to remove.

    3. If in doubt of having selected the correct sequence, verify it by selecting View video. See View video in an incident project.

  3. Select Remove and confirm the removal.
