Отключить справку

Для предотвращения доступа операторовПрофессиональный пользователь приложения клиента XProtect. к функции справки можно отключить справку. В этом случае при нажатии клавиши F1 ничего не будет происходить, а контекстные ссылки и кнопки Справка не будут видны. При необходимости справку можно снова включить.

Требования

Администратор также может контролировать доступность справки на стороне сервера. Справку можно включать и отключать только в случае, если администратор системы не заблокировал данный параметр.

Действия:

  1. Перейдите в окно Параметры. Будет открыта вкладка Приложение.

  2. В списке Онлайн-справка выберите Недоступна.
  3. Закройте окно.
  4. Чтобы убедиться в том, что справка отключена, нажмите F1. Ничего не должно произойти.
