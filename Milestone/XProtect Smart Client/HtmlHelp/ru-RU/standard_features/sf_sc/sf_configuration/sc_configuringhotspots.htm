Области внимания (настройка)

Перед началом работы с областью вниманияОпределенная позиция в видах XProtect Smart Client, используемая для просмотра изображений с камеры с увеличением или высоким качеством. необходимо:

  1. Добавление области внимания в вид. См. раздел Добавить области внимания в виды.
  2. Задать параметры областей внимания. См. раздел Настройки области внимания.

Добавить области внимания в виды

Если в вашем виде есть область внимания и вы нажмете камеру, видеопоток с камеры будет показан в высоком разрешении в элементе вида область вниманияОпределенная позиция в видах XProtect Smart Client, используемая для просмотра изображений с камеры с увеличением или высоким качеством..

Действия:

  1. Нажмите кнопку Настройка для перехода в режим настройки.
  2. На панели Обзор системы перенесите элемент Область внимания в нужное место вида. В соответствующем месте вида будет показан значок области внимания: .
  3. Снова нажмите кнопку Настройка для выхода из режима настройки.
  4. (Необязательно) Для настройки свойств области внимания откройте панель Свойства в режиме настройки.

Для экономии трафика можно задать низкое качество изображения для других позиций на виде, а для области внимания задать высокое качество.

Настройки области внимания

На панели Свойства (см. Параметры камер) можно задать параметры области вниманияОпределенная позиция в видах XProtect Smart Client, используемая для просмотра изображений с камеры с увеличением или высоким качеством.. Параметры Динамический поток, Качество изображения, Частота кадров и Сохранять пропорции изображения применяются ко всем камерам в области внимания.

Для просмотра свойств нужно выбрать элемент представления и нажать кнопку Настроить.
