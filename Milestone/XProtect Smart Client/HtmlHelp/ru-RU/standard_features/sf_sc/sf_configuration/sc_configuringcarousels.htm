  1. Добавление карусели в вид.

Добавить карусели в виды

КаруселиКонкретная позиция для просмотра видео с нескольких камер, одна за другой, в виде XProtect Smart Client. дают возможность постоянно переключаться между камерами в произвольном темпе.

  1. Нажмите кнопку Настройка для перехода в режим настройки.
  2. На панели Обзор системы перенесите элемент Карусель в элемент просмотра.
  3. В окне Настройка карусели:
    1. Перейдите в раздел Камеры.
    2. Дважды щелкните на каждой камере, которую нужно добавить в карусель.
  4. Чтобы определить последовательность появления камер в карусели, в списке Выбранные камеры переместите камеры вверх или вниз.
  5. Введите время отображения каждой камеры в карусели (в секундах). Можно задать значение сразу для всех камер или индивидуально для каждой.
  6. Нажмите кнопку ОК, чтобы закрыть окно Настройка карусели.
  7. Снова нажмите кнопку Настройка для выхода из режима настройки. Ваши изменения сохранены.
  8. Необязательный шаг: для изменения параметров карусели в режиме настройки откройте панель Свойства и нажмите кнопку Настройка карусели.

Изменение параметров карусели

В параметрах карусели можно добавлять и удалять камеры, менять порядок камер в карусели, а также регулировать параметры времени.

  1. Нажмите кнопку Настройка для перехода в режим настройки.
  2. Выберите элемент вида с помощью карусели.
  3. Слева прокрутите вниз до панели Свойства.
  4. Щелкните Настройка карусели. Появится новое окно.
  5. Внесите необходимые изменения и нажмите OK.
  6. Снова нажмите кнопку Настройка для выхода из режима настройки. Ваши изменения сохранены.

Параметры Прямая передача, Качество изображения, Частота кадров и Сохранять пропорции изображения на панели Свойства применяются ко всем камерам в карусели.
