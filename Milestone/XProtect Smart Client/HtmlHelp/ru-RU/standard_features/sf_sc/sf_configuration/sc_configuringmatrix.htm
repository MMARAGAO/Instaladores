Matrix (настройка)

Добавление Matrix в виды

Для отправки видео получателю MatrixПродукт, доступный в качестве дополнения к системам наблюдения XProtect. С XProtect Incident Manager вы можете документировать и управлять инциденты в XProtect Smart Client. в режиме реального времени сначала нужно добавить элемент Matrix в вид. Оператор может отправить видео получателю Matrix только из вида.

  1. В режиме настройки на панели Обзор системы перенесите элемент Matrix в ту область вида, где должен быть показан контент Matrix. Вокруг позиции в виде появится голубая рамка, показывая, что позиция содержит Matrix.
  2. Если выбрать позицию Matrix, можно задать ее свойства на панели Свойства.

    При просмотре видео реального времени или записанного видео можно дважды щелкнуть позицию Matrix (или любую другую позицию камеры в виде), чтобы развернуть его. В развернутом состоянии видео с камер с позиции Matrix отображается по умолчанию в полном качестве, независимо от выбранного качества изображения. Если нужно убедиться, что выбранное качество изображения также применяется в развернутом виде, установите флажок Сохранять при полноэкранном режиме.

  3. Повторите это для всех добавляемых позиций Matrix.

В режиме настройки на панели Свойства (см. Параметры камер) можно задать параметры позиций MatrixПродукт, доступный в качестве дополнения к системам наблюдения XProtect. С XProtect Incident Manager вы можете документировать и управлять инциденты в XProtect Smart Client..

Индекс окна

Для изменения позиции Matrix выберите другой номер. Возможно выбрать значение из диапазона, соответствующего числу позиций Matrix вида. 1 — основная позиция, в которой всегда отображается видео из последнего события, 2 — отображает видео из предыдущего события, 3 — отображает видео из события, произошедшего перед событием в позиции 2 и так далее.

Позволяет задать порт TCPTransmission Control Protocol; протокол (т. е. стандарт), используемый для пересылки пакетов данных по сети. Протокол TCP часто используется в сочетании с протоколом IP. Эта комбинация, известная как TCP/IP, позволяет длительное время передавать пакеты данных между двумя точками в сети, и она используется при соединении компьютеров и других устройств по Интернету. и пароль для передачи Matrix-видео с XProtectсервера VMS в представление XProtect Smart Client. Эта возможность доступна только при выборе позиции Matrix1; прочие позиции Matrix наследуют параметры соединения у позиции 1. По умолчанию для Matrix используется порт TCP 12345. Номер порта и пароль можно узнать у администратора системы.
