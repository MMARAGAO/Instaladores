LPR на вкладке Наблюдение (объяснение)

Во вкладке Наблюдение можно просматривать видео в реальном времени с камер, настроенных на распознавание номерных знаков (LPR). Доступен одновременный просмотр видеозаписей с нескольких камер LPR в одном виде. События LPR отображаются справа в виде при возникновении соответствия. В режиме настройки можно изменить параметры, определяющие отображение списка номерных знаков транспортных средств.

При нажатии на номерной знак в списке событий LPR видео в реальном времени автоматически приостанавливается и начинается самостоятельное воспроизведение видеозаписи. Для возврата к просмотру видео в реальном времени снова нажмите на номерной знак или щелкните значок Независимое воспроизведение в панели инструментов камеры.

Распознавание номерных знаков на вкладке «Поиск» (описание)

На вкладке Поиск можно выполнять поиск видеозаписей, связанных с транспортными средствами.

Вкладка LPR (объяснение)

На вкладке LPR можно расследовать события LPRСокращенное название технологии распознавания номерных знаков (LPR). со всех камер LPR, а также просматривать сопутствующие видеозаписи и информацию о распознанных номерных знаках. Регулярно обновляйте списки соответствий и создавайте отчеты.

На вкладке расположен список событий распознавания номерного знака (LPR) и предварительный просмотр камер для распознавания номерных знаков. В предварительном просмотре можно просматривать видео, связанное с событием LPR. Под областью предварительного просмотра отображается информация о номерном знаке и данные из списка соответствия номерных знаков и стиля знака, с которым связан этот номер.

Можно отфильтровать список событий по периоду, модулю страны, камере LPR, списку и стилю соответствия номерных знаков. Используйте поле Поиск регистрационного номера, чтобы найти определенный номерной знак. По умолчанию этот список отображает события LPR за прошедший час. См. также Список событий LPR (объяснение).

Можно отобрать данные об интересующих вас событиях и экспортировать их в виде отчета в формате PDF.

Кроме того, можно обновлять существующие списки соответствия с помощью функции Список соответствия номерных знаков.

Список событий LPR (объяснение)

Список событий LPRСокращенное название технологии распознавания номерных знаков (LPR). содержит данные обо всех событиях LPR. По умолчанию список отображает события LPR за прошедший час с новыми событиями вверху, но ваш системный администратор может изменить эту настройку.

Если выбрать событие LPR в списке, справа отображается предварительный просмотр видеоэпизода этого события. В строке заголовка предварительного просмотра показано название камеры LPR, с которой поступило событие LPR. Также можно просматривать:

  • Номер регистрационного знака

  • Модуль страны

  • Время события

  • Список соответствия, который активировал событие

  • Стиль регистрационного знака (см. Стили номерных знаков)

Способ отображения событий LPR в списке можно изменить: можно сортировать столбцы и перемещать их в разные положения. Можно отфильтровать события LPR с помощью фильтров в верхней части списка или воспользоваться полем Поиск регистрационного номера для поиска.

Список событий LPR отображает только данные, актуальные на момент поиска или фильтрования. Для получения новой информации о событиях LPR нажмите кнопку Обновить.

Стили номерных знаков

Стиль регистрационного знака — набор характеристик регистрационного знака, в том числе:

  • Размер и форма знака

  • Формат и шрифт текста

  • Цвета

  • Тип транспортного средства, где используется знак

Ваш системный администратор может группировать стили регистрационных знаков и задавать группе пользовательское название.

Можно добавлять в списки соответствия только те стили регистрационного знака, которые были сгруппированы и для которых было задано название администратором системы.

Фильтр событий LPR (объяснение)

Существует несколько способов фильтрации списка событий LPR и вывода только тех данных, которые вас интересуют. Нажмите на любой фильтр в верхней части списка для просмотра данных только о событиях LPR, связанных с этим фильтром. Применение любых фильтров мгновенно отражается на содержимом списка.

  • Период: Выберите один из доступных временных интервалов для просмотра событий LPR.
  • Модуль страны: Очистите или выберите модули страны, чтобы отобразить только события LPR, связанные с номерным знаком из определенной страны, области или конкретного региона
  • Камера LPR: Выберите одну или несколько доступных камер LPR, чтобы отобразить события LPR только с этих камер
  • Список соответствия Выберите один или несколько списков соответствия номерных знаков, чтобы отобразить события LPR только из этих списков

  • Стиль номерного знака: Выберите один или несколько стилей регистрационного знака для просмотра только тех событий LPR, которые с ними связаны

Фильтры можно объединять, например, по модулю страны и определенной дате.

Можно также воспользоваться полем Поиск регистрационного знака, чтобы найти определенный номерной знак. Введите сочетание символов для получения результатов поиска, в которых встречается данное сочетание. Например, если ввести символы XY 12, будет показан список номерных знаков, в которых есть и XY, и 12. Если вы укажете XY12, будут показаны только номерные знаки, в которых есть XY12.

Редактирование списков соответствия номерных знаков

Можно добавлять и удалять номерные знаки в списках соответствия номерных знаков.

  1. На вкладке LPR в правом верхнем углу окна нажмите элемент Списки соответствия, чтобы открыть диалоговое окно Списки соответствия.
  2. В окне Выбрать список соответствия выберите список для правки.

  3. Для добавления регистрационного номера знака или стиля регистрационного знака нажмите Добавить. Введите необходимую информацию и нажмите кнопку OK.

    Можно добавлять в списки соответствия только те стили регистрационного знака, которые были сгруппированы и для которых было задано название администратором системы.

  4. Чтобы править существующий знак, можно воспользоваться функцией поиска и найти необходимый регистрационный номер.
  5. Дважды нажмите строку, чтобы изменить ее, или выберите несколько строк и нажмите Правка.
  6. В диалоговом окне введите необходимые данные и нажмите кнопку OK. Если список соответствия содержит несколько столбцов, можно править информацию во всех полях.
  7. Чтобы удалить существующий знак, можно воспользоваться функцией поиска и найти необходимый регистрационный номер.
  8. При необходимости выберите несколько строк и нажмите кнопку Удалить.
  9. Нажмите кнопку Закрыть.

Можно также добавить регистрационный номер в список соответствия номерных знаков, правой кнопкой мыши щелкнув событие LPR, которого еще нет в списке, и выбрав пункт Добавить в список. Можно удалить номерной знак, выбрав соответствующее событие LPR и справа под областью предварительного просмотра нажав значок Удалить из списка.

Импорт и экспорт списков соответствия номерных знаков

Можно импортировать файл с необходимым списком номерных знаков в список соответствия номерных знаков. Доступны указанные ниже параметры импорта:

  • Добавление номерных знаков в существующий список
  • Замена существующего списка

Эта возможность может быть полезна в случае, например, управления списками из центрального местоположения. После этого можно поддерживать все локальные копии в актуальном состоянии, передавая файл.

Подобным образом можно экспортировать полный список номерных знаков из списка соответствия на внешний носитель.

  1. Для импорта списка соответствия:
    1. На вкладке LPR в правом верхнем углу окна нажмите элемент Списки соответствия, чтобы открыть диалоговое окно Списки соответствия.
    2. Выберите необходимый список.
    3. Чтобы импортировать файл, нажмите кнопку Импорт.
    4. В диалоговом окне укажите местоположение файла для импорта и тип импорта. Нажмите Далее.
    5. Подождите, пока отобразится подтверждение, и нажмите кнопку Закрыть.
  2. Для экспорта списка соответствия:
    1. Нажмите кнопку Экспорт.
    2. В диалоговом окне укажите местоположение файла для экспорта и нажмите Далее.
    3. Нажмите кнопку Закрыть.
    4. Можно открыть экспортированный файл, например, в программе Microsoft Excel, и внести в него правки.

Поддерживаются форматы .txt и .csv.

Экспорт событий LPR в виде отчета

Можно экспортировать отчет о событиях LPRСокращенное название технологии распознавания номерных знаков (LPR). в файл PDF.

  1. На вкладке LPR отфильтруйте или найдите события, которые должны быть включены в отчет.

    Если количество найденных событий очень велико, вы получите рекомендацию уточнить условия поиска и уменьшить число результатов.

  2. Нажмите на кнопку Отчет LPR.
  3. Укажите следующие данные и нажмите кнопку OK:
    • Имя отчета
    • Путь отчета
    • Поле комментария
    • Возможность добавить снимки

    Вверху справа в окне XProtect Smart Client будет показан индикатор хода выполнения.

  4. Нажмите кнопку Подробно, чтобы посмотреть отчет.

Если необходимо изменить формат бумаги или шрифт, откройте окно Настройки, выберите Дополнительно, и измените настройки Отчет в формате PDF или Шрифт в отчете в формате PDF.

LPR на вкладке Диспетчер тревог

На вкладке Диспетчер тревог можно просматривать и расследовать тревоги, связанные с LPR. Прежде чем просматривать информацию, необходимо выполнить настройку, как указано ниже:

  • Отображение элементов, связанных с LPR
  • Список сигналов тревоги должен быть в режиме событий (см. Просмотр данных, связанных с распознаванием номерных знаков LPR)

Ознакомьтесь с разделами, посвященными управлению тревогами, чтобы узнать больше о работе программы XProtect Smart Client.

Просмотр данных, связанных с распознаванием номерных знаков LPR

Данные, связанные с распознаванием номерных знаков LPR, можно видеть в списке тревог. При выборе событий в качестве источника данных отображаются все данные, связанные с распознаванием. При выборе тревог в качестве источника данных отображаются только связанные с распознаванием данные, которые соотнесены с тревогой.

Требования

Чтобы можно было воспользоваться полем Тип, упоминаемым на следующих шагах, это поле должно быть включено администратором в XProtect Management Client.

Действия:

  1. Перейдите к вкладке Диспетчер тревог.
  2. Нажмите кнопку Настройка для входа в режим настройки.
  3. Для просмотра связанных с распознаванием данных, которые соотнесены с тревогой:
    1. В списке Источник данных выберите Тревога.
    2. Снова нажмите кнопку Настройка для выхода из режима настройки. Ваши изменения сохранены. Связанные с распознаванием данные показаны в списке сигналов тревоги.
    3. Для просмотра входящих тревог LPR введите LPR в поле Тип.

  4. Для просмотра всех данных, связанных с распознаванием:
    1. В списке Источник данных выберите Событие.
    2. Снова нажмите кнопку Настройка для выхода из режима настройки. Ваши изменения сохранены. Связанные с распознаванием данные показаны в списке сигналов тревоги.
    3. Для просмотра всех входящих событий LPR введите LPR в поле Тип.

Отфильтрованные результаты отобразятся в списке тревог только после того, как вы выйдете из измененного поля.
