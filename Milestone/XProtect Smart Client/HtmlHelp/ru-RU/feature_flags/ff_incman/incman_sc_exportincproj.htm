Экспорт проекта с инцидентом

Эпизоды проекта с инцидентом удаляются с сервера записи, когда оканчивается заданное время хранения VMS XProtect. Если эпизоды нужны вам как доказательство на более долгий период времени, экспортируйте проект с инцидентом до окончания времени хранения.

Когда вы отправляете проект с инцидентом на экспорт, все эпизоды из проекта с инцидентом вставляются в Список экспорта на вкладке Экспорт. При экспортировании эпизодов из проекта с инцидентом вы можете использовать все настройки, доступные на вкладке Экспорт.

См. также Параметры экспорта.

Для экспорта эпизодов, добавленных в проект с инцидентом:

  1. Выберите вкладку Инциденты.
  2. Выберите проект с инцидентом, который вы хотите экспортировать.
  3. Нажмите Отправить на экспорт.

    4. Появится вкладка Экспорт, а все эпизоды из проекта с инцидентом будут добавлены в Список экспорта.

  4. Выберите предпочитаемый формат и другие настройки экспорта.

См. также Создание видеодоказательств.

При сохранении или отправке экспорта, как правило, сохраняется или отправляется отчет со всей текстовой информацией, добавленной в этот проект с инцидентом.
