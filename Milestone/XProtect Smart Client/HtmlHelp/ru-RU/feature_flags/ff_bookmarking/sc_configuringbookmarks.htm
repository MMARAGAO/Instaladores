Закладки (настройка)

Эта функция доступна только в некоторых XProtect системах VMS. Информация о возможностях системы приведена в разделе Различия систем наблюдения.

Включить подробные закладки

Для того чтобы иметь возможность задавать имена и описания закладок, а также менять стандартные период времени, необходимо включить подробные закладки.

Действия:

  1. Перейдите в окно Параметры.

  2. Выберите Функции.
  3. Чтобы включить подробные закладки для прямой передачи, выберите Добавить сведения о закладке в списке Закладка на вкладке Наблюдение.
  4. Чтобы включить подробные закладки для видео в записи, выберите Добавить сведения о закладке в списке Закладка на вкладке Воспроизведение.
  5. Нажмите кнопку Закрыть.
