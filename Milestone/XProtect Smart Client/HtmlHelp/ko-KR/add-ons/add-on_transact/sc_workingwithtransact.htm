XProtect Transact (사용량)

XProtect Transact감시 시스템의 추가 기능으로 사용할 수 있는 제품입니다. XProtect Transact을(를) 이용하면, 시간 연결된 POS 또는 ATM 트랜잭션 데이터와 비디오를 결합시킬 수 있습니다.이(가) 시스템에 구성된 경우, 라이브 트랜잭션을 관찰하고, 여러 방법으로 트랜잭션을 조사하며 트랜잭션을 인쇄할 수 있습니다.

XProtect Transact (개요)

이 주제는 XProtect Smart Client 에서 XProtect Transact감시 시스템의 추가 기능으로 사용할 수 있는 제품입니다. XProtect Transact을(를) 이용하면, 시간 연결된 POS 또는 ATM 트랜잭션 데이터와 비디오를 결합시킬 수 있습니다.을(를) 가지고 할 수 있는 작업에 대하여 개요를 설명합니다. 탭 단위로 기능을 설명합니다.

탭

설명
실시간

라이브 탭에서는 라이브 트랜잭션 및 트랜잭션을 모니터링하는 카메라의 감시 비디오를 관찰할 수 있습니다. 뷰에는 여러 개의 트랜잭션 뷰 항목이 포함될 수 있습니다. 여기서 최대 두 개의 카메라에서 들어오는 비디오 스트림과 동기화되어 화면에 영수증이 표시됩니다.

설정 모드에서 트랜잭션 뷰를 생성하고 수정합니다.
재생

재생 탭에서는 지난 트랜잭션 및 트랜잭션을 모니터링하는 카메라의 감시 비디오를 찾아볼 수 있습니다. 뷰에는 여러 개의 트랜잭션 뷰 항목이 포함될 수 있습니다. 여기서 최대 두 개의 카메라에서 들어오는 비디오 스트림과 동기화되어 화면에 영수증이 표시됩니다.

설정 모드에서 트랜잭션 뷰를 생성하고 수정합니다.
알람 관리자

알람 관리자 탭에서는 트랜잭션에 관련된 이벤트와 알람을 보고 조사할 수 있습니다. 이벤트는 이벤트 목록에 표시됩니다. 트랜잭션 이벤트를 그룹화하려면 트랜잭션을 이벤트 유형으로 필터링해야 합니다. 이벤트 목록에서 라인을 클릭하면 해당 이벤트와 관련된 비디오가 미리보기에 표시됩니다.
Transact

트랜잭트 탭에서, 자유 텍스트 검색을 수행하고 필터를 적용하여 트랜잭션을 조사할 수 있습니다. 트랜잭션 라인은 시간, 트랜잭션 소스 및 라인 이름을 기준으로 정렬할 수 있는 목록에 표시됩니다. 라인을 클릭하면 연결된 카메라의 관련 비디오 스틸 프레임이 표시됩니다. 미리보기 영역 아래에 영수증이 표시됩니다.

라이브 트랜잭션 관찰

카메라 레코딩과 트랜잭션을 통해 실시간 비디오 감시를 수행하면서 실시간 트랜잭션을 관찰할 수 있습니다. 예를 들어, 현금 등록기와 판매 직원을 관찰하면서 거래를 지속적으로 관찰해야 할 수 있습니다.

요구사항

트랜잭션 표시를 위한 뷰를 설정했습니다. 자세한 정보는 트랜잭션 보기 설정를 참조하십시오.

단계:

  1. 라이브 탭에서 뷰 창을 확장합니다.

  2. 트랜잭션에 대한 뷰 설정을 선택합니다. 진행되고 있는 트랜잭션이 있으면 영수증이 화면에 표시되고 연관된 카메라의 라이브 비디오가 표시됩니다.

트랜잭션 뷰 항목이 영수증보다 좁으면 가로 스크롤 막대가 나타나 숨겨진 영수증 부분을 볼 수 있습니다. 스크롤 막대에 접근하려고 하면 뷰 항목 도구 모음이 나타나 스크롤 막대를 가리게 됩니다. 스크롤 막대에 접근하려면 컨트롤(Ctrl) 키를 누른 상태에서 커서를 뷰 항목 영역으로 가져가십시오.

을/를 선택하여 영수증의 폰트 크기를 변경합니다.

트랜잭션 조사

뷰에서 트랜잭션 조사

트랜잭션을 조사하는 가장 간단한 방법은 비디오 레코딩과 동기화하여 화면에 영수증 롤이 표시되는 뷰에서 트랜잭션을 보는 것입니다.

요구사항

트랜잭션 표시를 위한 뷰를 설정했습니다. 자세한 정보는 트랜잭션 보기 설정를 참조하십시오.

단계:

  1. 재생 탭을 클릭합니다.

  2. 뷰 창에서 트랜잭션 뷰를 선택합니다. 뷰가 구성된 방식에 따라 트랜잭션 소스와 연결된 카메라와 함께 하나 이상의 영수증이 나타납니다.

  3. 뒤로 모드에서 비디오 시퀀스를 찾아보려면 시간 라인을 오른쪽으로 끕니다.
  4. 앞으로 모드에서 비디오 시퀀스를 찾아보려면 시간 라인을 왼쪽으로 끕니다.
  5. 또는 버튼을 사용하여 비디오를 앞이나 뒤로 재생합니다.

트랜잭션 뷰 항목이 영수증보다 좁으면 가로 스크롤 막대가 나타나 숨겨진 영수증 부분을 볼 수 있습니다. 스크롤 막대에 접근하려고 하면 뷰 항목 도구 모음이 나타나 스크롤 막대를 가리게 됩니다. 스크롤 막대에 접근하려면 컨트롤(Ctrl) 키를 누른 상태에서 커서를 뷰 항목 영역으로 가져가십시오.

을/를 선택하여 영수증의 폰트 크기를 변경합니다.

검색 및 필터를 사용하여 트랜잭션 조사

필터와 검색어를 사용하여 트랜잭션 및 연관된 비디오 레코딩을 조사할 수 있습니다. 필터를 사용하면 지난 7일 동안의 트랜잭션 또는 특정 현금 등록기의 트랜잭션 등 검색 범위를 좁힐 수 있습니다. 검색어를 이용하면 판매 직원의 이름이나 승인되지 않은 할인 등 트랜잭션의 특정 데이터를 쉽게 확인할 수 있습니다.

  1. 트랜잭션 탭을 클릭합니다.
  2. 오늘 드롭다운 목록에서 시간 간격을 선택합니다.

  3. 소스 드롭다운 목록에서 조사하려는 트랜잭션 소스를 선택합니다. 비활성화된 소스는 "(CashRegister_)"와 같이 "()"로 표시됩니다.

  4. 검색어를 입력합니다. 필터 아래에 검색 결과가 트랜잭션 라인으로 표시되고 영수증에서 검색 항목이 강조 표시됩니다.
  5. 목록을 업데이트하려면 을(를) 클릭합니다.
  6. 트랜잭션 라인을 클릭하여 연관된 비디오 스틸 프레임을 표시합니다. 또는 소스 드롭다운 목록에서 조사하려는 트랜잭션 소스를 선택합니다.

기본적으로, 트랜잭션 데이터는 30일 동안 저장되지만 구성에 따라 데이터를 최대 1000일까지 저장할 수 있습니다.

비활성화된 소스에서 트랜잭션 조사

시스템 관리자가 트랜잭션 소스를 비활성화했더라도 관련 비디오 녹화물과 함께 해당 소스에서 이전 트랜잭션을 볼 수 있습니다.

단계:

  1. 트랜잭션 탭을 클릭합니다.

  2. 모든 소스 드롭다운 목록에서 비활성화된 트랜잭션 소스를 선택합니다. 괄호는 소스가 비활성화되었음을 나타냅니다(예: "(CashRegister_1)").

  3. 지난 7일 과 같은 시간 간격을 선택하거나 사용자 정의 간격을 설정합니다.
  4. 을(를) 클릭하여 지정된 시간 간격에 대한 트랜잭션 라인을 봅니다.
  5. 트랜잭션 라인을 선택하여 정확하게 이 시점부터 관련 비디오의 정지 프레임을 봅니다.
  6. 또는 버튼을 사용하여 앞으로 또는 뒤로 재생 모드에서 비디오를 재생합니다.

기본적으로, 저장된 트랜잭션 데이터는 30일 후에 삭제됩니다. 그러나 시스템 관리자가 보존 기간을 1 ~ 1000일 범위 내의 특정 기간으로 변경했을 수 있습니다.

트랜잭션 이벤트 조사

예를 들어, 특정 품목이 구입된 트랜잭션을 확인하는 식으로 트랜잭션 이벤트를 조사할 수 있습니다. 트랜잭션 이벤트를 조사하면 알람 목록에서 이벤트 관련 정보와 관련 비디오 레코딩이 표시됩니다.

요구사항

트랜잭션 이벤트를 기준으로 필터링하려면, 유형 필드를 XProtect Smart Client 에 추가해야 합니다. 이 작업은 시스템 관리자만 수행할 수 있습니다.

단계:

  1. 알람 관리자 탭을 클릭합니다.
  2. 상단 오른쪽에서 설정 을 클릭하여 설정 모드로 들어갑니다.
  3. 속성 창을 확장합니다.
  4. 데이터 소스 목록에서 이벤트 를 선택하고 설정 을 다시 클릭하여 설정 모드를 나갑니다. 모든 이벤트는 목록에 표시되며 가장 최근 항목이 맨 위에 옵니다.

  5. 트랜잭션 이벤트만 보려면 필터 섹션을 확장하고 유형 필드에 트랜잭션 이벤트 를 입력합니다. 자동으로 필터가 적용되고 트랜잭션 이벤트만 목록에 나타납니다.

  6. 시스템 관리자가 정의한 특정 이벤트를 보려면 메시지 목록을 열고 이벤트를 선택합니다.
  7. 이벤트와 연관된 비디오 레코딩을 보려면 목록에서 이벤트를 클릭합니다. 미리보기 영역에서 비디오 재생이 시작됩니다.

트랜잭션 알람 조사

트랜잭션 이벤트에 의해 트리거된 알람을 조사할 수 있습니다. 알람은 알람 및 연관된 비디오 레코딩에 관한 정보를 볼 수 있는 알람 목록에 나타납니다.

요구사항

트랜잭션 이벤트를 기준으로 필터링하려면, 유형 필드를 XProtect Smart Client 에 추가해야 합니다. 이 작업은 시스템 관리자만 수행할 수 있습니다.

단계:

  1. 알람 관리자 탭을 클릭합니다.
  2. 상단 오른쪽에서 설정 버튼을 클릭하여 설정 모드로 들어갑니다.
  3. 속성 창을 확장합니다.
  4. 데이터 소스 목록에서 알람 을 선택하고 설정 을 다시 클릭하여 설정 모드를 나갑니다. 가장 최근 알람이 맨 위에 표시됩니다.
  5. 트랜잭션 이벤트에 의해 트리거된 알람만 보려면 필터 섹션을 확장하고 유형 필드에 트랜잭션 이벤트 를 입력합니다. 자동으로 필터가 목록에 적용됩니다.
  6. 특정 이벤트에 의해 트리거된 알람을 보려면 메시지 목록을 열고 이벤트를 선택합니다.
  7. 알람과 연관된 비디오 레코딩을 보려면 목록에서 알람을 클릭합니다. 미리보기 영역에서 비디오 재생이 시작됩니다.

트랜잭션 인쇄

트랜잭트 작업공간에서 트랜잭션을 보면서, 한 번에 하나씩 트랜잭션을 인쇄할 수 있습니다. 인쇄물에는 트랜잭션 라인에 해당하는 시간에 관련 카메라의 스틸 이미지와 영수증이 표시됩니다.

단계:

  1. 트랜잭션 탭을 클릭합니다.

  2. 트랜잭션 조사에서 설명된 대로 출력하기 원하는 트랜잭션을 찾습니다.

  3. 트랜잭션 아래에서 인쇄 를 클릭하여 인쇄합니다. Windows 대화 상자가 나타납니다.
  4. 필요한 프린터를 선택하고 확인 을 클릭합니다.
