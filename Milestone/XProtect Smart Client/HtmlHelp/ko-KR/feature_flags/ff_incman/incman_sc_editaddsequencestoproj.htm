사건 프로젝트에 시퀀스 추가

실시간으로 사건 프로젝트를 생성한 경우 일반적으로 사건이 시작되기 전에 발생한 작업을 보여 주는 시퀀스를 사건 프로젝트에 추가하려고 합니다.

한 번에 하나의 시퀀스 추가

  1. 라이브 또는 재생 탭에서 시퀀스를 추가할 카메라 위치의 오른쪽 하단 모서리에 있는 사건 프로젝트에 추가 아이콘을 선택합니다.

  2. 기존 사건 프로젝트에 추가를 선택합니다.
  3. 시작 시간 선택을 선택하고 사건이 시작된 위치를 찾을 때까지 타임라인을 드래그합니다.
  4. 다시 사건 프로젝트에 추가를 선택하고 계속해서 종료 시간을 선택합니다.
  5. 종료 시간 선택을 선택하고 사건이 종료된 위치를 찾을 때까지 타임라인을 드래그합니다.
  6. 다시 사건 프로젝트에 추가를 선택하고 OK를 선택합니다.
  7. 시퀀스를 추가할 사건 프로젝트를 선택합니다.

    8. 구체적인 사건 프로젝트를 찾기 어려운 경우 필터 옵션을 사용할 수 있습니다.사건 프로젝트 필터를 참조하십시오.

여러 시퀀스 추가

  1. 라이브 또는 재생 탭에서 시퀀스를 추가할 카메라 위치의 오른쪽 하단 모서리에 있는 사건 프로젝트에 추가 아이콘을 선택합니다.

  2. 초안 시퀀스 목록에 추가를 선택합니다.
  3. 시작 시간 선택을 선택하고 사건이 시작된 위치를 찾을 때까지 타임라인을 드래그합니다.
  4. 다시 사건 프로젝트에 추가를 선택하고 계속해서 종료 시간을 선택합니다.
  5. 종료 시간 선택을 선택하고 사건이 종료된 위치를 찾을 때까지 타임라인을 드래그합니다.
  6. 다시 사건 프로젝트에 추가를 선택하고 OK를 선택합니다.
  7. 사건 프로젝트에 대한 모든 관련 시퀀스를 얻기 위해 필요한 만큼 위의 단계를 반복합니다.
  8. 사건 프로젝트에 추가 아이콘을 선택하고 시퀀스 목록 초안 보기를 선택합니다.
  9. 기존 사건 프로젝트에 추가하려는 시퀀스 목록 초안에서 시퀀스를 선택하고 기존 사건 프로젝트에 모두 추가를 선택합니다.
  10. 시퀀스를 추가할 사건 프로젝트를 선택합니다.

    11. 구체적인 사건 프로젝트를 찾기 어려운 경우 필터 옵션을 사용할 수 있습니다.사건 프로젝트 필터를 참조하십시오.
