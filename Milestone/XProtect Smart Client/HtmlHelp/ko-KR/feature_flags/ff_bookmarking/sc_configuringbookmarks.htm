북마크(구성)

이 기능은 특정 XProtect VMS 시스템에서만 사용할 수 있습니다. 사용 중인 시스템의 기능에 관한 정보는 감시 시스템의 차이점를 참조하십시오.

상세 북마크 활성화

북마크에 이름과 설명을 부여하고 기본 시간 범위를 변경할 수 있도록 하려면 세부 정보를 활성화해야 합니다.

단계:

  1. 설정 창을 여십시오.

  2. 기능 탭을 선택하십시오.
  3. 라이브 비디오에 대한 자세한 북마크를 활성화하려면, 라이브 탭의 북마크 목록에서 북마크 세부 정보 추가 를 선택합니다.
  4. 레코딩된 비디오에 대한 상세 북마크를 활성화하려면 재생 탭의 북마크 목록에서 북마크 세부 정보 추가 를 선택합니다.
  5. 닫기 를 클릭합니다.
