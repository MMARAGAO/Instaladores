Milestone Interconnect (사용량)

타임라인 및 Milestone Interconnect (설명됨)

선택한 카메라가 Milestone Interconnect 설치의 일부이고 상호 연결된 장치일 경우, 선택한 카메라의 타임라인에 검색 데이터가 표시됩니다. 수행된 검색은 레코딩으로 표시됩니다. 타임라인에서 음영을 사용하여 레코딩을 포함하는 기간과 검색(에서 데이터 검색 Milestone Interconnect 참조)을 요청하기 위해 필요한 기간을 곧바로 식별할 수 있습니다.

  • 짙은 회색은 해당 시간에 대한 레코딩이 없음을 나타냅니다
  • 짙은 바둑판 패턴은 요청된 레코딩이 없고, 따라서 레코딩이 있는지 여부를 알 수 없음을 나타냅니다
  • 빨간색 음영은 레코딩이 있음을 나타냅니다
  • 옅은 바둑판 패턴은 데이터의 검색이 요청되었음을 나타냅니다

짙은 바둑판 패턴이 있는 타임라인은 요청된 레코딩이 없어 레코딩이 있는지 여부를 알 수 없음을 나타내고, 옅은 바둑판 패턴이 있는 타임라인은 비디오 검색이 요청되었음을 나타냅니다.

에서 데이터 검색 Milestone Interconnect

Milestone Interconnect™ 에서는 현재 VMS 시스템에 통합되어 있는 독립 VMS 시스템에 속한 하나 이상의 장치로부터 데이터를 검색할 수 있게 해줍니다. 일례는 중앙 사이트에 연결된 선박 또는 버스 모바일 설치입니다.

요구사항

XProtect Smart Client 이(가) Milestone Interconnect 설치의 일부이며 최소 한 대 이상의 카메라가 최신 저장 장치를 지원하고 사용자가 필요한 사용자 권한을 가지고 있습니다.

단계:

  1. 재생 탭의 타임라인에서 시간 선택 모드 버튼을 클릭하여 데이터를 검색할 시퀀스의 시작 및 종료 시간을 선택합니다.
  2. 도구 모음의 오른쪽에서 검색 을 클릭하여 검색 창을 엽니다.

  3. 관련 카메라를 선택하고 검색 시작을 클릭합니다.

    응용 프로그램 도구 모음에서 상태 버튼을 클릭하면 상태 창(상태 창(설명됨) 참조)에서 검색 작업의 진행률을 볼 수 있습니다.

  4. 진행 중인 검색 작업을 중지하려면 작업공간 영역의 상단에 있는 알림 영역 또는 진행 중인 작업 옆에 있는 상태 창에서 중지 를 클릭합니다.
