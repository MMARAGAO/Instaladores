자주 묻는 질문: 뷰

뷰를 설정하지 않고 즉시 비디오를 볼 수 있습니까?

예, 많은 XProtect Smart Client 사용자가 뷰를 먼저 설정할 필요 없이 자신의 XProtect Smart Client 에 있는 비디오를 즉시 볼 수 있습니다.

개인 뷰: 카메라가 몇 개 없는 기본 소규모 감시 시스템과 같이 특정 유형의 감시 시스템에 연결된 경우(감시 시스템의 차이점 참조), 감시 시스템은 시스템의 모든 카메라를 이용해 하나의 개인 뷰를 자동으로 생성할 수 있습니다. 이러한 뷰를 기본 뷰 라고 합니다. 기본 뷰에 대한 액세스 권한이 있는 경우, XProtect Smart Client 에 처음 로그인할 때 기본 보기가 자동으로 표시되므로 사용자의 XProtect Smart Client 에서 즉시 비디오 보기를 시작할 수 있습니다.

공유 뷰: 공유 뷰는 시스템 관리자나 다른 동료들에 의해 이미 생성되었을 수 있습니다. 공유 뷰가 이미 있고 해당 뷰에 대한 액세스 권한이 있는 경우, 사용자의 XProtect Smart Client 에서 즉시 비디오 보기를 시작할 수 있습니다.

내 뷰를 다시 만들어야 하는 이유는 무엇입니까?

경우에 따라 시스템 관리자가 감시 시스템에서 카메라 또는 사용자 속성을 변경할 수 있습니다. 이러한 변경 사항은 변경한 후 처음으로 로그인할 때 XProtect Smart Client 에 적용되므로 뷰를 다시 생성해야 하는 경우가 있습니다.

개인 또는 공유 뷰를 생성할 수 없는 경우에는 어떻게 합니까?

일반적으로 조직에서 공유 뷰를 생성하고 편집할 수 있는 사람은 적습니다. 시스템 관리자가 여러 개의 공유 뷰를 생성하여 유지 관리할 수 있습니다. 사용자가 로그인하면 공유 뷰를 자동으로 사용할 수 있게 되므로 해당 사용자가 추가 뷰를 생성할 필요가 없습니다.

액세스 권한을 가지고 있는 뷰를 어떻게 확인할 수 있습니까?

일반적으로 사용자가 공유 시스템에 대한 액세스 권한이 있는지는 시스템 관리자가 알려줍니다. 관리자가 알려주지 않는 경우 다음과 같은 방법을 통해 공유 뷰를 사용할 수 있는지 여부를 신속하게 파악할 수 있습니다.

라이브 또는 재생 탭의 뷰 창에는 항상 "개인"이라는 상위 수준의 폴더가 있습니다. 상위 수준 폴더인 개인 폴더는 개인 뷰에 액세스하는 데 사용되고, 해당 콘텐츠는 사용자가 생성한 뷰(있는 경우)에 따라 다릅니다.

뷰 창에 있는 다른 모든 상위 수준 폴더는 공유 뷰에 액세스하는 데 사용됩니다. 이러한 상위 수준 폴더의 이름은 구성된 내용에 따라 다릅니다.

뷰 창에 공유 뷰를 액세스하는 데 사용되는 하나 이상의 상위 수준 폴더가 포함되어 있다고 해서 공유 폴더를 실제로 사용할 수 있는 것은 아닙니다. 상위 수준 폴더에서 공유 뷰를 사용할 수 있는지 확인하려면 폴더를 확장합니다.

편집할 수 있는 뷰를 어떻게 확인할 수 있습니까?

폴더에 자물쇠 아이콘이 있는 경우 보호되고 있는 것이므로 해당 폴더에 새 뷰를 만들거나 기존 뷰를 편집할 수 없습니다.

다른 컴퓨터에서 내 뷰를 볼 수 있습니까?

해당 뷰에 대한 정보를 비롯한 사용자 설정이 감시 시스템 서버에 저장됩니다. 즉, 사용자 이름 및 암호를 사용하여 XProtect Smart Client 에 로그인하면 XProtect Smart Client 가 설치된 모든 컴퓨터에서 뷰(개인 및 공유)를 사용할 수 있습니다.

동작을 수행할 수 있는 권한이 없는 경우라도 해당 동작에 대한 오버레이 버튼을 추가할 수 있습니까?

예, 이렇게 하면 공유 뷰에서 버튼을 사용할 수 있으며, 사용자 자신에게 해당 버튼을 사용할 수 있는 권한이 없더라도 필요한 권한이 있는 동료는 버튼을 사용할 수 있습니다.

사용자에게 권한이 없는 동작의 버튼을 추가할 경우 설정 모드에서는 버튼이 흐리게 표시되고 라이브 탭을 사용할 경우에는 버튼이 표시되지 않습니다. 필요한 권한이 있는 동료는 라이브 탭에서도 해당 버튼을 사용할 수 있습니다.

오버레이 버튼을 추가한 후 내 권한이 변경되면 어떻게 합니까?

권한 변경은 사용자가 버튼을 사용할 수 있는 방식에 영향을 주며, 해당 동작에 대한 사용자 권한이 있는지 여부에 따라 버튼이 흐리게 나타나거나 사용할 수 있게 됩니다. 예를 들어, 수행할 권한이 없는 동작에 대해 버튼을 추가하고 필요한 권한을 갖도록 사용자 권한이 변경되면 해당 버튼이 사용 가능 상태로 변경됩니다.

오버레이 버튼을 어떻게 삭제합니까?

설정 모드에서 버튼을 마우스 오른쪽 버튼으로 클릭하고 삭제 를 선택합니다.

내보낸 비디오에 오버레이 버튼이 표시됩니까?

아니요, 비디오를 내보내는 경우 오버레이 버튼은 내보내기에 포함되지 않습니다.
