서버 연결의 상태 확인

상태 창에서 서버 연결 상태를 확인할 수 있습니다(예: 이전 보안 모델(HTTP) 또는 최신 보안 모델(HTTPS)을 사용하고 있는지 여부 확인). 다수의 사이트가 Milestone Federated Architecture 을(를) 통해 연결된 경우에는 연결된 사이트도 확인할 수 있습니다.

연결 상태

  • HTTPS

  • HTTP

  • 연결되지 않음

단계:

  1. 응용 프로그램 도구 모음에서 상태 버튼을 클릭합니다. 창이 표시됩니다.

  2. 로그인 정보 탭에서 연결 상태를 확인하십시오.

또한 상태 창(설명됨)를 참조하십시오.
