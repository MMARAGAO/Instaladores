사용자 권한(설명됨)

사용자 권한은 시스템 관리자가 중앙에서 지정하며, 이에 따라 특정 XProtect Smart Client 기능을 사용할 수 있는 권한이 결정됩니다.

기본적으로 시스템 관리자가 사용자 권한을 제한할 수 있습니다:

  • XProtect Smart Client 액세스
  • 각 탭에 액세스: 라이브, 재생, 알람 관리자 및 검색
  • 설정 모드
  • 특정 기능 사용
  • 보통 하나 이상의 카메라에서 비디오를 포함하는 뷰를 생성
  • 특정 카메라의 비디오 보기

XProtect Smart Client 의 기능을 사용할 수 있는 권한은 사용자마다 크게 다를 수 있습니다.

시간, 요일 등에 따라 사용자 권한이 다를 수 있습니다. 예를 들어 월요일~금요일의 특정 시간 동안 특정 카메라에서 비디오를 볼 수 있지만 이 시간 이외에는 비디오를 볼 수 없습니다.
