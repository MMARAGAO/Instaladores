도움말 비활성화하기

운영자XProtect 클라이언트 응용 프로그램의 전문가 사용자. 가 도움말 기능에 액세스하는 것을 방지하기 위해 도움말을 비활성화할 수 있습니다. 그다음 F1 을 눌러도 아무 일도 일어나지 않으면 맥락에 밀접한 링크와 도움말 버튼이 더 이상 보이지 않게 됩니다. 요청을 받은 경우 도움말 기능을 다시 활성화 할 수 있습니다.

요구사항

시스템 관리자가 서버측에서 도움말의 사용 가능성을 통제할 수 있습니다. 시스템 관리자가 이 설정을 잠그지 않았다면 도움말을 끄고 켜는 것만 가능합니다.

단계:

  1. 설정 창을 여십시오. 이는 응용 프로그램 탭에서 열립니다.

  2. 온라인 도움말 목록에서 사용 불가 를 선택합니다.
  3. 대화 닫기.
  4. 도움말 기능이 비활성화되었는지 확인하려면 F1 을 누르십시오. 아무 일도 일어나지 않아야 합니다.
