PTZ 프리셋(구성)

사용 중인 감시 시스템에 따라(감시 시스템의 차이점 참조), PTZ 프리셋을 생성, 편집 및 삭제할 수 있습니다.

PTZ 프리셋 추가

추가 PTZ 사전 설정조직의 직원, 자산, 운영, 서비스 또는 기능에 손상, 손실 또는 중단을 초래할 수 있는 부정적이거나 위험한 행위 또는 상황을 정의할 수 있습니다.

  1. 뷰에서 새 PTZ 프리셋을 제공하려는 관련 PTZ 카메라를 선택합니다.
  2. 카메라 도구 모음에서 PTZ 아이콘 을 클릭하여 PTZ 메뉴를 엽니다.
  3. PTZ 사전 설정 관리 를 클릭하여 창을 엽니다.

  4. 을(를) 클릭하여 새 프리셋 항목을 추가합니다.
  5. PTZ 사전 설정 항목을 선택하고 해당 PTZ 사전 설정에 대해 새 이름을 입력합니다.
  6. PTZ 버튼을 사용하여 필요한 위치로 이동한 다음 사전 설정 업데이트 를 클릭하여 저장합니다.
  7. 화살표를 사용하여 목록에서 PTZ 프리셋을 위나 아래로 이동합니다. 목록에 프리셋이 많이 있는 경우에 유용할 수 있습니다.

PTZ 프리셋 편집

사전 설정 위치를 변경하거나 이름을 바꾸는 등 기존 PTZ 사전 설정조직의 직원, 자산, 운영, 서비스 또는 기능에 손상, 손실 또는 중단을 초래할 수 있는 부정적이거나 위험한 행위 또는 상황을 변경할 수 있습니다.

  1. 뷰에서 수정할 PTZ 프리셋을 가진 PTZ 카메라를 선택합니다.
  2. 카메라 도구 모음에서 PTZ 아이콘 을 클릭하여 PTZ 메뉴를 엽니다.
  3. PTZ 프리셋 관리 를 클릭하고 대화 상자에서 PTZ 프리셋을 선택합니다.
  4. 프리셋 이름을 편집하려면 해당 PTZ 프리셋의 이름을 강조 표시합니다. 텍스트를 클릭하고 기존 이름을 덮어씁니다.

  5. 카메라가 올바른 위치에 있지 않은 경우, PTZ 버튼을 사용하여 필요한 위치로 이동한 다음, 프리셋 업데이트를 클릭하여 저장합니다.
  6. 위쪽 또는 아래쪽 화살표를 사용하여 목록에서 PTZ 사전 설정을 배열할 수 있습니다.
  7. 닫기 를 클릭합니다.

PTZ 프리셋 삭제

기존 프리셋을 삭제하려면 해당 프리셋을 선택하고 을(를) 클릭합니다.
