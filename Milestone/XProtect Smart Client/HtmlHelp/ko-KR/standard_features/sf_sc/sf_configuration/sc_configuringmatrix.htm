Matrix (구성)

뷰에 Matrix 추가

라이브 비디오를 MatrixXProtect 감시 시스템의 추가 기능으로 사용할 수 있는 제품입니다.XProtect Incident Manager을(를) 사용하면 XProtect Smart Client내에서 사건을 문서화하고 관리할 수 있습니다.-수신자에게 보낼 수 있으려면, 먼저 Matrix 항목을 뷰에 추가해야 합니다. 뷰 내에서만 운영자가 비디오를 Matrix-수신자에게 보낼 수 있습니다.

  1. 설정 모드의 시스템 개요 창에서 Matrix 요소를 Matrix 콘텐츠를 추가하려는 뷰의 위치로 끌어 옮깁니다. 뷰의 위치에 Matrix 콘텐츠가 있음을 나타내는 파란색 테두리가 표시됩니다.
  2. Matrix 위치를 선택하는 경우, 속성 창에서 해당 속성을 지정할 수 있습니다.

    라이브 또는 녹화 비디오를 시청하는 경우 Matrix 위치(또는 뷰의 다른 카메라 위치)를 두 번 클릭해서 최대화할 수 있습니다. 최대화 시 사용자가 선택한 이미지 화질 옵션과 관계없이 Matrix 위치에 있는 카메라의 비디오가 기본적으로 최대 품질로 표시됩니다. 최대화 시 선택한 이미지 화질도 적용하려면 최대화 시 유지 를 선택합니다.

  3. 추가하려는 각 Matrix 위치에 대해 반복하십시오.

Matrix 설정

설정 모드의 속성 (카메라 설정 참조) 창에서 MatrixXProtect 감시 시스템의 추가 기능으로 사용할 수 있는 제품입니다.XProtect Incident Manager을(를) 사용하면 XProtect Smart Client내에서 사건을 문서화하고 관리할 수 있습니다. 위치에 대한 설정을 지정할 수 있습니다.

창 색인

다른 숫자를 선택하여 Matrix 위치의 순위 지정을 변경합니다. 뷰에서 Matrix 위치의 수에 해당하는 범위에서만 숫자를 선택할 수 있습니다. 1 은 최신 이벤트의 비디오가 항상 표시되는 주 위치이고, 2 에는 이전에 감지된 이벤트의 비디오가 표시되고, 3 에는 2 위치의 이벤트 전에 감지된 이벤트의 비디오가 표시됩니다.
연결 설정...

Matrix 이(가) 트리거한 비디오를 XProtect VMS 서버에서 XProtect Smart Client 뷰로 전송하기 위해 TCPTransmission Control Protocol의 약어로서 네트워크에서 데이터 패킷을 보내는 데 사용되는 프로토콜(표준)입니다. TCP가 다른 프로토콜인 IP(Internet Protocol)와 함께 사용되는 경우가 있습니다. 이 조합(TCP/IP)을 사용하면 네트워크의 두 지점 간에 오랜 기간 동안 데이터 패킷을 보내고 받을 수 있으며 인터넷에 있는 컴퓨터 및 다른 장치를 연결할 때 사용됩니다. 포트 및 암호 를 지정할 수 있습니다. 이 기능은 Matrix 위치를 1 로 선택한 경우에만 사용할 수 있습니다. 나머지 Matrix 위치는 1 위치에 대해 지정된 연결 설정을 상속합니다. 기본적으로, Matrix 에 사용된 TCP 위치는 12345입니다. 사용할 포트 번호 또는 암호에 대한 사항은 시스템 관리자에게 문의하십시오.
