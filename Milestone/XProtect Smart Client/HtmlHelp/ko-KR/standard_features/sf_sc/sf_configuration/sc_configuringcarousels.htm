순환보기(구성)

순환보기XProtect Smart Client 의 뷰에서 하나씩 여러 카메라의 비디오를 보기 위한 특정 위치.를 사용하기 전에 다음을 반드시 수행하십시오:

  1. 뷰에 순환보기를 추가합니다.
  2. 순환보기에 원하는 카메라를 지정하여 추가하십시오.

뷰에 순환보기 추가

순환보기XProtect Smart Client 의 뷰에서 하나씩 여러 카메라의 비디오를 보기 위한 특정 위치.를 통해 순환보기 카메라 사이를 정의한 속도로 계속 탐색할 수 있습니다.

  1. 설정 을 클릭하여 설정 모드로 들어갑니다.
  2. 시스템 개요 창에서 순환보기 항목을 뷰 항목로 끌어다 놓습니다.
  3. 순환보기 설정 창에서:
    1. 카메라 섹션으로 이동합니다.
    2. 순환보기에 추가할 각 카메라를 찾아 두 번 클릭합니다.
  4. 순환보기에 카메라가 나타나는 순서를 결정하려면 선택한 카메라 목록에서 카메라를 위 또는 아래로 이동합니다.
  5. 각 카메라가 순환보기에 표시되는 시간(초)을 입력합니다. 모든 카메라 또는 각 카메라에 대한 값을 지정할 수 있습니다.
  6. 순환보기 설정 창을 닫으려면 확인 을 클릭합니다.
  7. 다시 설정 을 클릭하여 설정 모드를 종료합니다. 변경 사항이 저장됩니다.
  8. (선택 사항) 순환보기의 설정을 변경하려면 설정 모드의 속성 창으로 이동하여 순환보기 설정을 클릭합니다.

순환보기 설정 편집

순환보기 설정에서, 순환보기에서 카메라 추가 또는 제거, 카메라 순서 변경, 시간 설정 변경을 할 수 있습니다.

  1. 설정 을 클릭하여 설정 모드로 들어갑니다.
  2. 순환보기와 함께 뷰 항목을 선택합니다.
  3. 왼쪽 모서리에서 속성 창을 아래로 스크롤합니다.
  4. 순환보기 설정 을 클릭합니다. 창이 표시됩니다.
  5. 필요한 내용을 변경하고 확인 을 클릭합니다.
  6. 다시 설정 을 클릭하여 설정 모드를 종료합니다. 변경 사항이 저장됩니다.

속성 창의 라이브 스트림, 이미지 화질, 프레임 속도 및 이미지 가로 세로 비율 유지 설정이 순환보기에 있는 모든 카메라에 적용됩니다.
