XProtect Smart Client 설치

XProtect Smart Client 을(를) 사용하려면 컴퓨터에 설치해야 합니다. 감시 시스템 서버에서 XProtect Smart Client 을(를) 다운로드하여 컴퓨터에 설치할 수 있습니다.

Milestone XProtect 감시 시스템에 포함된 모든 새로운 특징과 기능에 액세스할 수 있도록 항상 최신 버전의 XProtect Smart Client 을(를) 사용할 것을 권장합니다.

  1. 브라우저를 열고 서버의 URL 또는 IP 주소를 사용하여 관리 서버에 연결하십시오.
  2. 다음 중 하나 입력:
    •   로컬 서버 (http://localhost/installation)
    •   원격 서버의 IP 주소(http://[IP_address]/installation)
  3. 시작 페이지에서 언어 를 클릭하고 사용할 언어를 선택합니다. XProtect Smart Client 설치 마법사가 시작됩니다.
  4. 마법사에서 설치 지침을 따르십시오. 마법사가 설치 경로를 추천해 줍니다. 일반적으로 추천된 설치 경로를 사용하면 됩니다. 그러나 이전에 추가 기능 제품을 사용한 적이 있는 경우, 이 경로가 더 이상 유효하지 않을 수 있습니다.
