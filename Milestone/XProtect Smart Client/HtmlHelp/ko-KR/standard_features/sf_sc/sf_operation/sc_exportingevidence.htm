비디오 증거물 생성

XProtect Smart Client 에서 사건 및 이벤트를 문서화하는 방법은 여러 가지가 있습니다. 예를 들어, 레코딩을 내보내고 비디오 스트림에서 단일 스틸 이미지를 만들 수 있습니다.

비디오 증거물을 잠궈 삭제되지 않도록 하거나 비디오를 잠금 상태로 내보낼 수 있습니다.

비디오, 오디오, 스틸 이미지 내보내기

비디오 증거물을 공유하려면 다양한 형식으로 비디오와 관련 오디오를 내보내기할 수 있습니다. XProtect VMS 시스템에 따라 사용 가능한 스틸 이미지와 기타 데이터 유형 또한 내보내기할 수 있습니다.

내보내기 탭에서, 다음과 같은 세 단계로 비디오 증거물을 내보내기할 수 있습니다.

  1. 내보내기하려는 비디오 시퀀스를 내보내기 탭 > 내보내기 목록 에 추가합니다. 또한 비디오 시퀀스를 내보내기 목록에 추가를 참조하십시오.

  2. 최소한 1개의 내보내기 형식을 선택하고 내보내기 설정을 조절합니다. 또한 내보내기 설정 조정를 참조하십시오.

  3. 내보내기 만들기. 또한 내보내기 생성하기를 참조하십시오.

비디오 시퀀스를 내보내기 목록에 추가

다음 위치에서 비디오 시퀀스를 내보내기 목록 에 추가할 수 있습니다.

내보내기 탭

내보내기 목록 에서, 항목 추가 를 선택하여 내보내기하려는 비디오 시퀀스를 추가합니다.

재생 탭

  1. 타임라인에서 버튼을 선택하여 내보내기하려는 시퀀스의 시작 및 종료 시간을 선택합니다(시간 탐색 제어(개요) 참조).

  2. 내보내기에 포함시킬 각 항목에 대해 관련 확인란 을 선택합니다.

  3. 내보내기 > 내보내기 를 선택하여 선택한 비디오 시퀀스를 내보내기 목록 에 추가하고 내보내기 탭으로 이동합니다.

    또는

    내보내기 > 내보내기 목록에 추가 를 선택하여 선택한 비디오 시퀀스를 내보내기 목록 에 추가하고 재생 탭에 유지합니다.

재생 탭 > 증거물 잠금 > 뷰 > 증거물 잠금 목록

  1. 증거물 잠금 목록 에서, 기존 증거물 잠금을 선택합니다.

  2. 내보내기 목록에 추가 를 선택하여 선택한 증거물 잠금을 내보내기 목록 에 추가하고 재생 탭에 유지합니다.

검색 탭

  1. 내보내기하고자 하는 각 검색 결과 위로 마우스를 올린 후 파란색 확인란 을 선택합니다.

  2. 파란색 동작 도구 모음에서 내보내기 목록에 추가 를 선택합니다.

내보내기 설정 조정

최소한 1개의 시퀀스를 내보내기 탭 > 내보내기 목록 에 추가한 후, 최소한 1개의 내보내기 형식을 선택해야 합니다. 선택적으로 내보내기 설정을 조정할 수 있습니다.

단계:

  1. 내보내기 목록 아래에서, 최소한 1개의 내보내기 형식을 선택합니다.

    • XProtect 형식으로 내보내기 - 내보내기와 함께 XProtect Smart Client – Player 을(를) 포함하려면 XProtect 형식을 사용하십시오. 다른 미디어 플레이어는 작동하지 않습니다. 수신자가 내보내기된 증거물이 조작되지 않았는지 확인할 수 있도록 하려면, 내보내기 설정 > XProtect 형식 > 디지털 서명 포함 을 선택합니다. 이렇게 하면 XProtect Smart Client – Player 의 서명 확인 버튼이 활성화됩니다.
    • 미디어 플레이어 형식으로 내보내기 - 대부분 미디어 플레이어가 재생할 수 있는 형식을 사용합니다. 이를 위해서는 내보내기가 조회되는 컴퓨터에 미디어 플레이어가 설치되어 있어야 합니다.
    • 스틸 이미지로 내보내기 - 선택한 기간에 대해 각 프레임의 스틸 이미지 파일을 내보냅니다.

  2. 선택적으로 내보내기 목록 상의 각 비디오 시퀀스에 대해, 시작 시간 및 종료 시간 을 변경할 수 있습니다.

  3. 선택적으로 비디오 시퀀스에 사생활 보호를 추가하여 다른 비디오 영역을 가릴 수 있습니다. 또한 내보내기를 하는 중 녹화 영역 보호를 참조하십시오.

  4. 선택적으로 각 포맷에 대해 내보내기 설정 을 변경할 수 있습니다. 또한 내보내기 설정를 참조하십시오.

내보내기 기간과 카메라 수는 내보내기를 완료하는 데 걸리는 시간에 영향을 줍니다.소요 시간을 줄이기 위해 내보내기 형식을 변경할 수 있습니다.

내보내기 생성하기

최소한 1개의 시퀀스를 내보내기 탭 > 내보내기 목록 에 추가한 후, 최소한 1개의 내보내기 형식을 선택한 후에 내보내기를 생성할 수 있습니다.

단계:

  1. 내보내기 버튼을 선택합니다. 내보내기 생성 창이 열립니다.
  2. 내보내기 이름 필드에 내보내기 이름이 자동으로 생성됩니다. 이 이름을 변경할 수 있습니다.
  3. 내보내기 대상 필드에서 내보내기 경로를 지정합니다. 사용자가 생성하는 내보내기는 여기에 지정된 폴더에 저장됩니다.
  4. 증거물을 내보내기 하려면 내보내기 생성 을 선택합니다.
  5. 내보내기는 내보내기 대상 으로 지정한 폴더에 생성 및 저장됩니다. 또한 내보낸 비디오 보기를 참조하십시오.

사용자 권한이 없으면 비디오 데이터를 내보내지 못할 수 있습니다.

내보내기를 하는 중 녹화 영역 보호

비디오를 내보낼 때 사생활 보호를 추가하여 선택 영역을 가릴 수 있습니다. 다른 사람이 비디오를 볼 때 사생활 보호 영역은 진한 블록으로 나타납니다.

여기 추가하는 사생활 보호는 현재 내보내기 및 내보내기 목록 에서 선택한 비디오에만 적용됩니다. 내보내기는 이미 시스템 관리자가 특정 카메라에 대해 정의한 사생활 보호를 포함하고 있을 수 있습니다. 또한 사생활 보호(설명됨)를 참조하십시오.

단계:

  1. 내보내기 탭 > 내보내기 목록 에서, 사생활 보호를 추가하려는 비디오 시퀀스를 선택합니다.
  2. 미리보기 창 위에서 사생활 보호 버튼을 클릭합니다. 사생활 보호 추가 창이 열립니다.
  3. 보호하고자 하는 각 영역에 대해 버튼을 클릭하고 보호하려는 하는 영역 위로 포인터를 끌어다 놓습니다.

  4. 사생활 보호의 일부를 해제하려면 버튼을 클릭하고 해제하려는 보호 영역 위로 포인터를 끌어다 놓습니다. 마스크 해제하려는 각 부분에 대해 이 단계를 반복합니다.

    일시적으로 보호를 숨기려면 보호 숨기기 버튼을 클릭한 채로 누르고 계십시오.

  5. 확인 을 클릭하여 내보내기 탭으로 돌아갑니다.

미리보기 이미지에는 셀이 있는 눈에 보이지 않는 그리드가 있습니다. 선택하는 영역에 어떤 셀의 일부분이 포함되어 있으면 시스템이 전체 셀을 가립니다. 그 결과 의도한 것보다 약간 더 많은 이미지를 가리게 될 수 있습니다.

스토리보드(설명됨)

스토리보드 기능은 하나의 카메라 또는 여러 카메라에서 비디오 시퀀스를 받아 그 모두를 하나의 결합된 흐름으로 붙여 넣는 기능입니다. 이벤트의 시퀀스, 즉 스토리보드를 내부 조사용 증거물이나 소송 자료로 사용할 수 있습니다.

관련이 없는 모든 시퀀스는 건너뛸 수 있으므로 필요하지 않은 장기간의 비디오 시퀀스를 검토하느라 시간을 낭비할 필요가 없습니다. 또한 관련 비디오를 포함하지 않는 시퀀스를 저장하여 저장 공간을 낭비할 필요도 없습니다.

스토리보드 내보내기

비디오 시퀀스를 하나의 결합된 흐름으로 붙여 넣은 후 내보내기함으로써 스토리보드를 만들 수 있습니다.

단계:

  1. 재생 탭에서 스토리보드에 추가하고자 하는 항목을 포함한 뷰를 엽니다.
  2. 타임라인에서 을(를) 클릭합니다.
  3. 스토리보드에 대한 시작 시간과 종료 시간을 선택합니다. 시간 탐색 제어(개요)를 참조하십시오.
  4. 추가하고자 하는 뷰에서 각 항목에 대해 해당 확인란 을 선택하고 내보내기 > 내보내기 목록에 추가 를 클릭합니다.

    스토리보드에 필요한 모든 항목을 추가할 때까지 1~4 단계를 반복합니다.

  5. 내보내기 과정을 계속 진행합니다. 내보내기 설정 조정 및 내보내기 생성하기를 참조하십시오.

잠긴 비디오 증거물 내보내기

증거물 잠금삭제되지 않도록 보호된 비디오 시퀀스.을 내보내기할 때, 해당 내보내기에는 카메라에 관련된 장치의 데이터도 포함됩니다.

단계:

  1. 재생 탭을 클릭합니다.
  2. 타임라인 오른쪽에서 증거물 잠금을 클릭하고 보기를 선택합니다.
  3. 재생 탭 대신 라이브 탭을 유지하려면 응용 프로그램 도구 모음에서 상태 버튼 을 클릭하고 증거물 잠금 목록을 선택합니다.

  4. 증거물 잠금을 선택하고 내보내기 목록에 추가 를 클릭합니다.

  5. 내보내기 과정을 계속 진행합니다. 내보내기 설정 조정 및 내보내기 생성하기를 참조하십시오.

내보낸 비디오 보기

생성한 내보내기는 내보내기 생성 창 > 내보내기 대상 필드에 지정한 폴더에 저장됩니다.

단계:

  1. 만든 직후에 내보낸 비디오를 보려면:

    1. 상단 우측 가장자리에 있는 XProtect Smart Client 에서, 내보내기 상세 내용 을 선택합니다.

      2. 내보내기 상세 내용 창 > 내보내기 위치 필드에서, 링크가 출력 폴더의 위치를 표시해 줍니다.

    2. 링크를 클릭하여 출력 폴더를 열고 내보내기된 파일에 액세스합니다.

  2. 이전 시점에서 비디오를 내보낸 경우:
    1. 내보내기 파일을 저장한 폴더로 이동합니다. 기본 위치는 C:\Users\<username>\Documents\Milestone\Video Export입니다. 해당 폴더의 위치는 내보내기 생성 창 > 내보내기 대상 필드에서 확인할 수 있습니다. 이 내용은 항상 동일한 내보내기 대상을 사용하는 경우에만 적용됩니다.
    2. 출력 형식에 따라 관련 폴더를 열고 비디오 파일이나 정지 이미지를 두 번 클릭합니다. 형식이 XProtect 형식 인 경우, 확장자가 .exe인 Smart Client – Player 파일을 두 번 클릭합니다.

감시 보고서 인쇄 또는 생성

필요에 따라 감시 카메라에서 얻은 스틸 이미지에 기반한 긴급 감시 보고서를 인쇄하거나 컴퓨터에 저장할 감시 보고서를 생성할 수 있습니다.

또한 알람 보고서 인쇄 및 알람 통계 받기를 참조하십시오.

단일 카메라에서 보고서 인쇄

라이브 카메라 또는 레코딩된 비디오에서 단일 스틸 이미지와 관련 정보를 인쇄할 수 있습니다. 추가한 노트 또한 인쇄됩니다.

단계:

  1. 저장된 스틸 이미지를 인쇄하려면 재생 탭으로 이동합니다.
  2. 라이브 스틸 이미지를 인쇄하려면 라이브 탭으로 이동합니다.
  3. 관심있는 카메라가 포함된 뷰를 엽니다.

  4. 항목 보기 위로 마우스를 이동합니다. 카메라 도구 모음이 나타납니다.

  5. 아이콘을 클릭하십시오. 창이 표시됩니다.

  6. 필요한 경우 메모를 추가합니다.
  7. 인쇄 를 클릭합니다. Windows 인쇄 대화 상자가 나타납니다.
  8. 필요한 경우, 인쇄 설정을 변경 후 인쇄하십시오. 아니면 그저 인쇄 를 클릭합니다.

또한 조직에서 알람 처리 기능을 사용하는 경우, 알람에 관한 정보를 인쇄할 수 있습니다. 또한 알람(설명됨)를 참조하십시오.

검색 결과에서 보고서 생성

검색 결과에 따라 이벤트 또는 인시던트에 관한 정보가 담긴 감시 보고서를 생성할 수 있습니다(예: 스틸 이미지, 이벤트 시간, 카메라 정보, 노트). 보고서는 PDF 파일로 저장됩니다.

단계:

  1. 검색 탭으로 이동하여 검색을 수행합니다.
  2. 보고서에 포함하고자 하는 각 검색 결과 위로 마우스를 올린 후 파란 체크 상자 를 선택합니다.
  3. 파란 동작 표시줄에서 을(를) 클릭합니다. 창이 표시됩니다.
  4. 기본 보고서 이름을 의미있는 이름으로 변경합니다. 보고서에서 이름은 페이지 상단에 표시됩니다.
  5. 보고서가 저장되는 폴더를 변경하려면 보고서 대상 위치 섹션에서 을(를) 클릭하고 다른 폴더를 선택합니다.
  6. 노트를 추가하려면 노트 필드에 텍스트를 입력합니다.

  7. 생성 을 클릭합니다. 진행 상태 표시줄을 통해 보고서가 언제 생성되었는지 알 수 있습니다.

  8. 보고서를 열려면 상세 내용 을 클릭하고 열기 를 클릭합니다. 대상 폴더를 열려면 링크를 클릭합니다.

보고서의 레이아웃을 변경하려면 설정 대화상자를 열고 고급 을 클릭한 후 PDF 보고서 형식 목록에서 다른 값을 선택합니다.

이미지를 클립보드로 복사

선택한 카메라에서 단일 스틸 이미지를 복사할 수 있습니다. 그런 다음 복사된 이미지를 워드 프로세서, 전자메일 클라이언트 등과 같은 다른 응용 프로그램에 비트맵 이미지로 붙여넣을 수 있습니다. 한 번에 하나의 카메라에서 단일 이미지만 복사할 수 있습니다.

  • 이미지를 복사하려면 카메라 도구 모음에서 클립보드로 복사 아이콘을 클릭하십시오.
