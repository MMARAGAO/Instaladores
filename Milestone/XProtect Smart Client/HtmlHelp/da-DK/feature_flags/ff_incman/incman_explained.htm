XProtect Incident Manager (explained)

Milestone has developed add-on products that fully integrate with XProtect to give you extra functionality. Your XProtect license file controls the access to add-on products.

Available functionality depends on the system you are using. See the complete feature list, which is available on the product overview page on the Milestone website (https://www.milestonesys.com/solutions/platform/product-index/).

XProtect Incident Manager er et Milestone-tilføjelsesprodukt, som hjælper organisationer med at dokumentere incidenter og kombinere dem med sekvenser af bevismateriale (video og eventuelt lyd) fra deres XProtect VMS.

Brugere af XProtect Incident Manager kan gemme alle deres incident-oplysninger i incident-projekter. Fra incident-projekterne kan de holde styr på status for hvert incident og de tilknyttede aktiviteter. På denne måde kan brugerne på en effektiv måde administrere incidenter og let dele solidt bevismateriale forbundet med incidentet, både internt med kolleger og eksternt med myndighederne.

Med XProtect Incident Manager kan organisationer på overblik og forståelse for de incidenter, der finder sted i de områder de overvåger. Med den viden kan man gennemføre tiltag, som minimerer risikoen for fremtidige gentagelser af lignende incidenter.

I XProtect Management Client kan organisationens XProtect VMS-administratorer definere, hvilke incident-egenskaber, der skal være tilgængelige i XProtect Incident Manager, så det passer til organisationens behov. XProtect Smart Client-operatørerne starter, gemmer og administrerer incident-projekter og tilføjer forskellige oplysninger til incident-projekterne. Herunder fri tekst, incident-egenskaber, som administratorerne har defineret og sekvenser fra XProtect VMS. For at få fuld sporbarhed logfører XProtect VMS, når administratorer definerer og redigerer incident-egenskaber, og når operatører opretter og opdaterer incident-projekterne.

Se også manualen for XProtect Incident Manager (https://doc.milestonesys.com/2022r2/da-DK/portal/portal/htm/chapter-page-incman.htm).
