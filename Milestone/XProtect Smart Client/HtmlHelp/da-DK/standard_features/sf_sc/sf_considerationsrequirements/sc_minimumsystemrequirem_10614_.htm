Minimumsystemkrav

Oplysninger om systemkravene for de forskellige VMS- og systemkomponenter kan du finde på Milestone-websitet (https://www.milestonesys.com/systemrequirements/).

Se oplysninger om dit system

For at se oplysninger om dit system, for eksempel operativsystem, DirectX-version og installerede enheder og drivere:

  1. Åbn menuen Start og skrive dxdiag.

  2. Klik på teksten dxdiag for at åbne vinduet DirectX Diagnostic Tool.

  3. På fanen System kan du se systemoplysninger.
