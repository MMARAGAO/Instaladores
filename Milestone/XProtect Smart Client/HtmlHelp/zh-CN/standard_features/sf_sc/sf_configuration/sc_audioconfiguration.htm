音频（配置）

音频设置

可以监听与正在查看的视图或摄像机无关的记录音频。 必须在播放选项卡的导航功能中指定时间才能确定要监听哪些记录音频。

名称

说明
麦克风

选择您希望从其收听音频的麦克风。

如果麦克风列表显示无麦克风硬件，则说明您的计算机没有播放监控系统音频所需的硬件。这通常是因为计算机没有安装声卡。如果列表显示无麦克风来源，则连接到摄像机的麦克风不可用。
静音

选择静音麦克风或扬声器。
扬声器

选择要通过其讲话的扬声器。

如果扬声器列表显示无扬声器硬件，则说明您的计算机没有播放监控系统音频所需的硬件。这通常是因为计算机没有安装声卡。如果列表显示无扬声器来源，则连接到摄像机的扬声器不可用。

如果监控系统的多个摄像机连接有扬声器（并且拥有访问它们的必要权限），从扬声器列表中选择所有扬声器可同时通过所有连接的扬声器讲话。
讲话

要讲话时，单击并按住鼠标按键。
电平指示器

电平指示器表示说话的音量。如果音量很低，则需要靠近麦克风或在 Windows 中调整您的音频设置。如果电平指示器显示没有任何音量，则检查该麦克风是否已连接并且设置正确。
锁定选择的音频设备

选择摄像机或视图时，默认也会选中对应的麦克风和/或扬声器。但是，如果需要特定摄像机的音频而不考虑正在查看的内容，则您可以选择锁定选择的音频设备。

示例：假设您不但需要通过与摄像机 A 相连的麦克风和扬声器监听犯罪受害者讲话并对其讲话，同时又迫切需要查看视图中不同位置上显示的摄像机 X、Y 和 Z。通过选择锁定选择的音频设备，便可在查看其他摄像机的同时通过摄像机 A 与受害者通信。
仅列出来自当前视图的设备

如果监控系统中包含大量麦克风和/或扬声器，则在音频窗格中选择麦克风和扬声器所使用的列表可能会很长。为避免这种情况，通过选择仅列出来自当前视图的设备，可以限制列表只包含与当前视图有关的麦克风和扬声器。

在此上下文中，当前视图还包括您作为浮动窗口打开以及主显示器和副显示器上的任何视图（请参阅 多重窗口或显示器（已解释））。
