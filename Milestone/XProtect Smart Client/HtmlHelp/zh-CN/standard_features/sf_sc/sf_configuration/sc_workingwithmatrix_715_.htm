Matrix（使用率）

此功能仅适用于特定的 XProtect 视频管理软件系统。有关系统中的功能的信息，请参阅 监控系统的差别。

Matrix（已作说明）

Matrix 是一种功能，使您能够接收来自任意监控系统摄像机的视频，或将其发送到网络上的任意监视器（称为 Matrix 接收方）。在发生定义的事件，例如侦测到移动时，或其他用户要共享重要的实时视频时，通常的 Matrix 配置将自动显示所需的 Matrix 接收方上的实时视频。只要在监控系统服务器上配置了 Matrix，就可以在 Matrix 视图中包含 XProtect Smart Client 内容。在发生特定事件或其他用户要与您共享视频时，实时视频将自动显示在您的 Matrix 视图中。

查看 Matrix 内容（已解释）

MatrixXProtect 监控系统的附加产品。使用 XProtect Incident Manager，您可以记录和管理 XProtect Smart Client 中的事件。 设置中使用的事件在监控系统上发生的预定义事件；监控系统将其用于触发动作。根据监控系统的配置，外置传感器的输入信号、侦测到移动、收到其他应用程序的数据或用户的手动输入都可能会触发事件。事件的发生可能触发各种动作，例如让摄像机按特定的帧速率进行记录、触发输出、发送电子邮件或这些动作的组合。或摄像机完全取决于监控系统服务器的 Matrix 配置，或者取决于其他用户要与您共享的内容。不能在 XProtect Smart Client 内对此进行控制。但是，可以根据需要向视图中的多个位置添加 Matrix 内容，从而可以同时观看来自多个 Matrix 触发的实时视频。

Matrix 位置在工具栏上与 Matrix 图标一起显示：。通过双击 Matrix 可将其最大化。

一个视图可包含多个 Matrix 位置。这样，就可以同时观看来自多个 Matrix 触发的实时视频。如果视图包含多个 Matrix 位置，这些位置总是按顺序进行排列；其中一个位置为第一 Matrix 位置，另一个为第二位置，依此类推。收到第一个 Matrix 触发的实时视频流时，第一 Matrix 位置会自动显示实时视频流。收到下一个 Matrix 触发的视频流时，会运用先入先出的原则：接收的上一视频流会转移到视图中的第二 Matrix 位置，而第一 Matrix 位置则显示最新的视频流，依此类推。Matrix 位置的顺序是自动排列的：添加的第一个 Matrix 位置是第一 Matrix 位置，添加的下一个位置是第二位置，依此类推。您可以在设置模式下更改此排名。

在播放选项卡上，Matrix 位置显示该 Matrix 位置在实时选项卡中上次使用的摄像机视频。您可以使用播放选项卡的导航功能播放此视频。

将视频手动发送到 Matrix 接收方

您无法将视频发送到热点（请参阅 热点（已解释））或轮巡显示（请参阅 轮巡显示（已解释））。

要求

Matrix 内容已添加到视图。请参阅 将 Matrix 添加到视图。

  1. 选择视图。
  2. 在摄像机工具栏上，单击更多 > Matrix，然后选择相关 Matrix 接收方。
