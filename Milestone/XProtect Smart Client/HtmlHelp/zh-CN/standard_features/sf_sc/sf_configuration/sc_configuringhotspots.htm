热点（配置）

在您可以使用热点XProtect Smart Client 视图的一个特殊位置，用于查看放大的和/或高质量的摄像机图像。之前，您必须：

  1. 将热点添加至视图。 请参阅 将热点添加到视图。
  2. 指定热点设置。 请参阅 热点设置。

将热点添加到视图

如果您的视图包含热点，并且您单击了摄像机，则摄像机的视频馈送在热点XProtect Smart Client 视图的一个特殊位置，用于查看放大的和/或高质量的摄像机图像。视图项目中以高分辨率显示。

步骤：

  1. 单击 设置 进入设置模式。
  2. 在系统概览窗格内，单击并将热点项目拖动到视图中的所需位置。该位置会显示热点图标：.
  3. 再次单击设置以退出设置模式。
  4. （可选）要设置热点的属性，请在设置模式下，转到属性窗格。

要节省带宽，您可以为视图中的其他位置指定低图像质量，为热点指定高图像质量。

热点设置

在属性（请参阅 摄像机设置）窗格中，可指定热点XProtect Smart Client 视图的一个特殊位置，用于查看放大的和/或高质量的摄像机图像。的设置。实时数据流、图像质量、帧速率和保持图像纵横比设置应用于热点中的所有摄像机。

要显示属性，必须选择视图项目，然后单击设置。
