将 Matrix 添加到视图

要将实时视频发送至 MatrixXProtect 监控系统的附加产品。使用 XProtect Incident Manager，您可以记录和管理 XProtect Smart Client 中的事件。 收件人，首先您必须将 Matrix 项目添加到视图。只有在视图内才可以将视频发送至 Matrix- 收件人。

  1. 设置模式中，在系统总览窗格中，将 Matrix 元素拖动至您希望添加 Matrix 内容的视图位置。蓝色边线出现，表示视图中该位置具有Matrix内容。
  2. 当选择 Matrix 位置时，您可以在属性窗格中指定它的属性。

    查看实时视频或记录的视频时，可双击 Matrix 位置（或视图中的任何其他摄像机位置）以将其最大化。最大化后，Matrix 位置的摄像机的视频将默认以完全质量显示，而不考虑选择的图像质量。如果希望确保最大化时选择的图像质量也适用，请选择最大时保持相同图像质量。

  3. 对想要添加的各 Matrix 位置重复以上步骤。
