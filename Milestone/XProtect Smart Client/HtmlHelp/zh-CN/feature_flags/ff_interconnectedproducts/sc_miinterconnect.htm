时间轴和 Milestone Interconnect（已解释）

如果所选的摄像机是 Milestone Interconnect 安装的一部分，并且是互连设备，那么所选摄像机的时间轴会显示检索数据。已经发生的检索会显示为记录。在时间轴上，您可以通过底纹快速识别哪些时间段包含记录以及您需要为哪些时间段请求检索（请参阅 从 检索数据Milestone Interconnect）。

  • 深灰色表示该时间没有记录
  • 深色棋盘图案表示未请求任何记录，因此不知道是否存在记录
  • 红色底纹表示存在记录
  • 浅色棋盘图案表示已经请求对数据进行检索

时间轴上的深色棋盘图案表示未请求任何记录，因此不知道是否存在记录，浅色棋盘图案表示已经请求对视频进行检索。

从 检索数据Milestone Interconnect

Milestone Interconnect™ 让您可以从属于与当前视频管理软件系统集成的独立视频管理软件系统的一个或多个设备中检索数据。 一个例子是与中央站点连接的船舶或公共汽车上的移动安装。

要求

XProtect Smart Client 是 Milestone Interconnect 安装的一部分，并且至少一台摄像机支持边缘存储，同时您具备必要的用户权限。

步骤：

  1. 在播放选项卡的时间轴中，单击时间选择模式 按钮来选择希望检索数据片段的开始和结束时间。
  2. 在工具栏的右侧，单击检索打开检索窗口。

  3. 选择相关摄像机，然后单击开始检索。

    您可以通过单击应用程序工具栏上的状态按钮，在状态窗口中查看检索作业的进度（请参阅 状态窗口（已作说明））。

  4. 要停止正在进行的检索作业，请单击工作区顶部通知区域中，或状态窗口中，正在进行的作业旁边的停止。
