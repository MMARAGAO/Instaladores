Aktualisieren XProtect Smart Client

Während der Anmeldung, wenn Sie eine ältere Version von XProtect Smart Client verwenden als der Server, zu dem Sie eine Verbindung aufbauen, informiert Sie eine Meldung darüber, dass eine neue Version des XProtect Smart Client zur Verfügung steht, und wo Sie die neue Version der Software herunterladen können. Milestone empfiehlt Ihnen, die neue Version herunterzuladen.

Wenn XProtect Smart Client neuer ist, als der Server, mit dem Sie sich verbinden, stehen bestimmte Funktionen ggf. nicht zur Verfügung.

Anzeigen von Versions- und Plug-in-Informationen

Die genaue Versionsnummer Ihres XProtect Smart Client zu kennen, kann wichtig sein, wenn Sie Hilfe vom Support benötigen oder ein Upgrade vornehmen möchten. Gegebenenfalls müssen Sie auch wissen, welche Plug-ins Ihr XProtect Smart Client verwendet.

Die Version von XProtect Smart Client kann sich auch darauf auswirken,mit welcher XProtect-Serverversion es kompatibel ist.

Schritte:

  1. Öffnen Sie XProtect Smart Client.

  2. Klicken Sie in der Symbolleiste auf Hilfe > Über. Ein Fenster wird angezeigt.
