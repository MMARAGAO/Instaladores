Hotspots (Konfiguration)

Bevor Sie den HotspotEine besondere Position zum Anzeigen vergrößerter und/oder hochwertiger Kamerabilder in XProtect Smart Client-Ansichten., verwenden können, müssen Sie:

  1. Hinzufügen des Hotspots zu einer Ansicht. Siehe Hotspots zu Ansichten hinzufügen.
  2. Geben Sie die Einstellungen für den Hotspot an. Siehe Hotspot-Einstellungen.

Hotspots zu Ansichten hinzufügen

Wenn Ihre Ansicht einen Hotspot enthält, und Sie auf eine Kamera klicken, wird der Videofeed von der Kamera mit hoher Auflösung im Ansichtselement HotspotEine besondere Position zum Anzeigen vergrößerter und/oder hochwertiger Kamerabilder in XProtect Smart Client-Ansichten.angezeigt.

Schritte:

  1. Klicken Sie auf Setup, um in den Setup-Modus zu wechseln.
  2. Klicken Sie im Bereich Systemübersicht auf das Hotspot-Element und ziehen Sie es an die gewünschte Position in der Ansicht. Beachten Sie, dass die Position nun ein Hotspot-Symbol anzeigt: .
  3. Klicken Sie erneut auf Setup, um den Setup-Modus zu verlassen.
  4. (Optional) Um die Eigenschaften für den Hotspot festzulegen, gehen Sie im Setup-Modus zum Eigenschafts-Fensterbereich.

Um Bandbreite zu sparen, können Sie für die anderen Positionen in Ihrer Ansicht eine niedrige Bildqualität festlegen, sowie eine hohe Qualität für den Hotspot.

Hotspot-Einstellungen

Im Bereich Eigenschaften (siehe Kamera-Einstellungen) können Sie die Einstellungen für den HotspotEine besondere Position zum Anzeigen vergrößerter und/oder hochwertiger Kamerabilder in XProtect Smart Client-Ansichten. angeben. Die Einstellungen für Live Stream, Bildqualität, Bildfrequenz und Bildseitenverhältnis beibehalten gelten für alle Kameras im Hotspot.

Damit die Eigenschaften angezeigt werden, müssen Sie das Anzeigeobjekt auswählen und dann auf Einrichtung klicken.
