Mindestsystemanforderungen

Informationen zu den Systemanforderungen der verschiedenen Komponenten und Anwendungen Ihres Systems finden Sie auf der Milestone Website (https://www.milestonesys.com/systemrequirements/).

Informationen über Ihr System anschauen

Um Informationen zu Ihrem System anzuzeigen, wie zum Beispiel das Betriebssystem und die Version von DirectX und die installierten Geräte und Treiber:

  1. Öffnen Sie das Startmenü und geben Sie ein dxdiag.

  2. Klicken Sie auf den Text dxdiag um das Fenster DirectX-Diagnosetool zu öffnen.

  3. Sehen Sie auf der Registerkarte System die Systeminformationen.
