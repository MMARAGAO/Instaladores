Den Status Ihrer Serververbindung überprüfen

Im Status-Fenster können Sie den Status Ihrer Serververbindung überprüfen, z. B. um festzustellen, ob Sie ein älteres (HTTP) oder das neueste Sicherheitsmodell (HTTPS) verwenden. Wenn mehrere Standorte über Milestone Federated Architecture verbunden sind, können Sie auch die verbundenen Standorte prüfen.

Verbindungsstatus

  • HTTPS

  • HTTP

  • Keine Verbindung

Schritte: 

  1. Klicken Sie auf der Anwendungssymbolleiste auf die Schaltfläche Status. Ein Fenster wird angezeigt.

  2. Überprüfen Sie auf der Registerkarte Anmeldeinformationen den Status Ihrer Verbindung.

Siehe auch Statusfenster (Erklärung).
