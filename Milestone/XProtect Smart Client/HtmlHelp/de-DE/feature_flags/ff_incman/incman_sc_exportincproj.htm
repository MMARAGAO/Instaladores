Exportieren eines Vorfallprojekts

Die Sequenzen in einem Vorfallprojekt werden vom Aufzeichnungsserver gelöscht, wenn die in Ihrem XProtect VMS festgelegte Speicherzeit abläuft. Falls Sie die Sequenzen für längere Zeit als Beweismaterial benötigen, exportieren Sie das Vorfallprojekt vor dem Ablauf der Speicherzeit.

Wenn Sie ein Vorfallprojekt zum Export senden, werden alle Sequenzen aus diesem Projekt in die Exportliste auf der Registerkarte Exporte eingefügt. Sie können alle auf der Registerkarte Export verfügbaren Einstellungen verwenden, wenn Sie die Sequenzen aus dem Vorfallprojekt exportieren.

Siehe auch Exporteinstellungen.

Zum Export von Sequenzen, die einem Vorfallprojekt hinzugefügt wurden:

  1. Wählen Sie die Registerkarte Vorfälle aus.
  2. Wählen Sie das Vorfallprojekt aus, das Sie exportieren möchten.
  3. Wählen Sie Zum Exportieren senden aus.

    4. Die Registerkarte Export wird angezeigt und alle Sequenzen aus dem Vorfallprojekt werden zur Exportliste hinzugefügt.

  4. Wählen Sie die gewünschten Formate und sonstige Einstellungen für Ihren Export aus.

Siehe auch Erstellen von Videobeweisbildern.

Wenn Sie Ihren Export speichern oder freigeben, sollten Sie normalerweise auch einen Bericht mit allen Informationen in Textform, die dem gleichen Vorfallprojekt hinzugefügt wurden, speichern oder freigeben.
