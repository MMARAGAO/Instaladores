Aggiornamento XProtect Smart Client

Durante l'accesso, se si utilizza una versione precedente di XProtect Smart Client del server a cui ci si connette, un messaggio informa che è disponibile una nuova versione dell'applicazione XProtect Smart Client, indicando anche dove scaricare la nuova versione del software. Milestone consiglia di scaricare la nuova versione.

Se XProtect Smart Client è più recente del server a cui ci si connette, alcune funzioni potrebbero non essere disponibili.

Visualizzazione della versione e delle informazioni sui plugin

È importante conoscere l'esatta versione di XProtect Smart Client di cui si dispone quando si richiede assistenza o si desidera eseguire un aggiornamento. In questi casi, è necessario conoscere i plug-in utilizzati dalla propria versione di XProtect Smart Client.

La versione di XProtect Smart Client può anche determinare la compatibilità con la versione del server XProtect.

Operazioni:

  1. Avviare XProtect Smart Client.

  2. Nella barra degli strumenti, fare clic Guida > Informazioni su. Viene visualizzata una finestra.
