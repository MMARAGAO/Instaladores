Disabilitare la Guida

Per impedire agli operatoriUn utente professionale di un'applicazione client XProtect. di accedere alla funzione di aiuto, è possibile disattivare la guida. Poi non succede nulla quando si preme F1 e i link sensibili al contesto e i pulsanti Guida di aiuto non sono più visibili. Se necessario, è possibile abilitare nuovamente la Guida.

Requisiti

La disponibilità della Guida può anche essere controllata dal server dall'amministratore di sistema. È possibile disabilitare o abilitare la guida solo se l'amministratore di sistema non ha bloccato questa impostazione.

Operazioni:

  1. Aprire la scheda Impostazioni. Si apre sulla scheda Applicazione.

  2. Nell'elenco Guida online, selezionare Non disponibile.
  3. Chiudere la finestra di dialogo.
  4. Per verificare che la Guida sia stata disabilitata, premere F1. Non dovrebbe succedere nulla.
