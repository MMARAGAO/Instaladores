Controllo dello stato della connessione al server

Nella finestra Stato, è possibile controllare lo stato della connessione al server, ad esempio, per vedere se si sta utilizzando un modello di sicurezza più vecchio (HTTP) o il modello di sicurezza più recente (HTTPS). Se più siti sono connessi tramite Milestone Federated Architecture, è possibile anche controllare i siti connessi.

Stati di connessione

  • HTTPS

  • HTTP

  • Non connesso

Operazioni:

  1. Sulla barra degli strumenti dell'applicazione, fare clic sul pulsante Stato. Viene visualizzata una finestra.

  2. Nella scheda Informazioni sull'accesso, controllare lo stato della connessione.

Consultare anche Finestra Stato (spiegazione).
