Bir olay projesine ek diziler eklemek

Gerçek zamanlı olarak bir olay projesi oluşturduysanız, genellikle olay başlamadan önce neler olduğunu gösteren olay projenize dizileri eklemek istersiniz.

Her seferinde bir dizi eklemek

  1. Live (Canlı) veya Playback (Oynat) sekmesinde, bir dizi eklemek istediğiniz kameranın sağ alt köşesinde Add to incident project (Olay projesini ekle) simgesini seçin.

  2. Add to existing incident project (Seçileni mevcut olay projesine ekle) öğesini seçin.
  3. Select start time (Başlangıç zamanını seç) ve olayın başladığı yeri buluncaya kadar zaman çizelgesini sürükleyin.
  4. Bitiş zamanını seçmeye ilerlemek için tekrar Add to incident project (Olay projesine ekle) öğesini seçin.
  5. Select end time (Bitiş zamanını seç) ve olayın bittiği yeri buluncaya kadar zaman çizelgesini sürükleyin.
  6. Tekrar Add to incident project (Olay projesine ekle) öğesini seçin ve OK (Tamam) öğesini seçin.
  7. Dizi eklemek istediğiniz olay projesini seçin.

    8. Belirli bir olay projesini bulmak zorsa, filtre seçeneklerini kullanabilirsiniz. Bkz. Olay projeleri için filtre.

Çok sayıda diziler eklemek

  1. Live (Canlı) veya Playback (Oynat) sekmesinde, bir dizi eklemek istediğiniz kameranın sağ alt köşesinde Add to incident project (Olay projesini ekle) simgesini seçin.

  2. Add to draft sequence list (Taslak dizi listesine ekle) öğesini seçin.
  3. Select start time (Başlangıç zamanını seç) ve olayın başladığı yeri buluncaya kadar zaman çizelgesini sürükleyin.
  4. Bitiş zamanını seçmeye ilerlemek için tekrar Add to incident project (Olay projesine ekle) öğesini seçin.
  5. Select end time (Bitiş zamanını seç) ve olayın bittiği yeri buluncaya kadar zaman çizelgesini sürükleyin.
  6. Tekrar Add to incident project (Olay projesine ekle) öğesini seçin ve OK (Tamam) öğesini seçin.
  7. Olay projeniz için ilgili tüm dizileri almak için yukarıdaki adımları mümkün olduğunca çok kez tekrarlayın.
  8. Add to incident project (Olay projesine ekle) simgesini seçin ve View draft sequence list (Taslak dizi listesini görüntüle) öğesini seçin.
  9. Mevcut olay projenize eklemek istediğiniz taslak dizi listesinde dizileri seçin ve Add all to existing incident project (Tümünü mevcut olay projesine ekle) öğesini seçin.
  10. Dizi eklemek istediğiniz olay projesini seçin.

    11. Belirli bir olay projesini bulmak zorsa, filtre seçeneklerini kullanabilirsiniz. Bkz. Olay projeleri için filtre.
