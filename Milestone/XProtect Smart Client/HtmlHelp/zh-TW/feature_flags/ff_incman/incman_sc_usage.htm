XProtect Incident Manager (使用量)

當您的視訊管理軟體上安裝並配置了 XProtect Incident Manager 加值產品，且視訊管理軟體的管理員向您提供了所需的權限時，您可以在 XProtect Smart Client 中記錄和管理事件，在事件專案中儲存所有資訊。

在事件標籤上，您可以檢視和更新您的事件專案，並管理您的事件。您通常會新增片段和各種文字資訊到您的事件專案，以記錄和證明在事件發生時發生了什麼。

如果要改進事件管理，您可以：

  • 篩選事件專案清單以快速找到相關的事件專案
  • 查看不同事件專案的狀態、類型、開始及結束時間，以及說明
  • 保持以事件管理的最新操作和其他進展更新您的時間專案
  • 產生報告並匯出事件專案的片段，以在內部或外部分享證據
